Generate an input format (FMT)

This allows you to generate additional input sheets which can be selected by using the drop-down Worksheet found at the top of the ABCD window  at the time of inputing information.

To create a new input format:

Select the fields that are going to form part of the input sheet from the list. In order to select a field you can:

  • Double click on the respective field
  • Click on a field, and then to extend the selection,  use the keys [Shift]  or  [Ctrl-Shift] and then click on the arrow
  • To copy all the fields to the input sheet, click on

To remove fields from the input sheet:

  • Double click on the respective field
  • Click on a field and then, to extend the selection, use the keys [Shift]  or  [Ctrl-Shift] and then click on the arrow
  • To remove all the fields from the input sheet, click on

The extracted fields from the input sheet will be shown at the bottom of the window containing the list of fields

Use the buttons  or  to position the fields in your input sheet.

Use the link  to preview the input format process.

When the format is complete, it should be saved including the field name where the file is stored, and a brief description of the format generated. The file name of the format  may only contain letters, numbers and the character _. In addition the file name should always begin with a letter. Spaces are not accepted  nor are special characters. The  description must be reasonably short so that it can be seen correctly in the  Worksheet menu located in the top of the main  ABCD window.

To modify an existing entry format

Select the format you want to edit from the dropdown menu, then click on  Edit. Fill in the format information in the Selected Fields window. Follow the instructions above to add or delete format fields.

To delete an existing entry format

Select the format you want from the drop down menu and then click on Remove to delete.

Formats are stored in the folder def in the database directory, with the extension .fmt. The list of existing formats is kept in the file formats.wks. Each storage format is separated by the character | and contains a line for the name and description.

Example of a generated format:

H||Main entrances|0|0|||||||||||
100
110
111
240
242
245
246
250
260
300

As can be seen, data entry sheets include the style of the FDT H (heading) or L (dividing line) and the labels of the selected fields. This means that the changes carried out  in the FDT and the subsequent generation of the data entry sheets, will be present when the data entry sheet is selected.

The file  formats.wks  is read to populate the drop-down  Worksheet menu.When creating or deleting a format, the drop-down  Worksheet menu included in the main window menu is automatically updated.
