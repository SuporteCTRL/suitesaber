List of Terms

The alphabetical list of terms contains a list of the terms that have been identified as the Main Entry in the database`s FDT.

This list is built with the extracted terms of the inverted list, using the prefix and the extraction field format identified in the FDT.

In this alphabetical list you can:

  1. Open the list of terms that start with a specific character

  2. Continue the display of terms starting from the last one that has been shown in the list (option More terms)

  3. Write the root of a term and after pressing Enter scroll to the most similar term

When clicking on any of the entries shown in the lists, the record is opened in the editing window.

This window allows you to change the database from which you are copying the records and provide a conversion format to enable compatibility between the structures of database records.
