Create a database by copying the existing structures in Winisis

A database can be created by uploading a backup of the FDT, FST and PFT in a local definition under Winisis.

ABCD allows you to upload the FDT from the local machine and convert it to the FDT structure maintained under this system.

Similarly you can upload the FST and an output format (PFT).

Note that the selected output format must be written using pre and post literals in order to reproduce an HTML page correctly. Display formats which contain elements of the formatting language defined for viewing in WinIsis and not in a browser, will result in a syntax error when displayed under ABCD.

Once these three steps are completed the ABCD database will be created and will be added to the list of existing databases.

So that the database can be operated by other users who do not have permission the "System Administrator", needs to enter the option User Management, to create or select users who can access the database and give them the proper permissions.
