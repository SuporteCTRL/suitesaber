Configuring List of Variables for Statistics

This option enables you to define database variables which you can use for the extraction of statistics.

It requires three columns of data:

Field Define the title of the field to display in the rows and columns of the table of frequency or occurrence

Extraction Format		 The pft will be used to extract the contents of the field in the database in order to mirror it in the table of frequency or occurrence. The value produced by the PFT will be used as the heading of the corresponding row or column which will be added to by the action of totalling
 Prefix  (Yet to be implemented) To allow the extraction of variables directly from the inverted file, reducing the time taken to read the data

This file is named stats.cfg and is placed in the folder def of the corresponding database.

The variables defined here can be used for two types of process:

  • For output of tables of occurence or tables of frequency, selecting the variables going to rows and the variables going to columns
  • For preparing pre-defined tables from which the results will be extracted by selecting the name from the corresponding box
