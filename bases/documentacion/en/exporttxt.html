Exporting the database to a TXT file with tags

To export a database in the form of a TXT file with tags, the conversion tables already defined in the Import option are used.

To export follow this procedure:

  1. Select the conversion table you will use during the process. To do that, press the option Select that appears in each conversion table shown on the screen

  2. Select the records that you want to export using one of the options available:

    • Mfn range

    • Search

    • A letter of the alphabet

  1. Select the option you want to use to obtain the TXT file with the exported records, using the two options available:

  2. Word processor

  3. Previous view

Once you select the records to export, insert the name of the ISO file you will create into the box and click on the icon.

When the ISO file is created, a window with the results and a link to the new ISO file will appear. If you want to download the file, right click on the link to see the option to do so.

Afterwards, during the import operation, the ISO file you obtained will be shown as part of the list of files stored in the folder ./bases/wrk.

Important note: If the database has many records, it is advisable to use the MX utility to create the ISO file.
