Types of material

This table describes the policies to be applied to the different types of material available in the database. The type of material referred to in the table must be generated by the extraction format applied to the database of lendable materials; this output should contain, amongs other data, the type of material to be located in the table, in order to assign the respective policy.

If the extraction format does not produce any output, then, the loans operator will determine the type of loan to apply selecting one option from the table Loans types.

When the loan register is saved, so is the type of material or type of loan it has been assigned, in order to properly apply return/fine/suspension policies.
