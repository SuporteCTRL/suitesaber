Prior to using this option you must configure the variables of the database involved in the statistics and, optionally, frequency and/or contingency tables where applicable.

To set the variables involved in statistical processes, use the option Edit database definition of the menu at the beginning of the administration module

Generation of statistics

Using an existing table

A drop-down menu with the tables previously defined in the option is displayed Edit database definition. Select the table you wish to use

Create a table

This displays the list of fields of the database that have been selected to obtain statistics. Select the fields to use for the table rows and, optionally, the fields to use for the columns.

Tip: select rows for the field that possess the greater diversity of values.

Generate output

This section indicates the method of selection of records for which you wish to obtain a frequency matrix or contingency.

  • By a range of Mfn
  • By searching

If the selection is By searching the Advanced search form will be presented in a popup window, to assist in building the search terms to apply to the database for the extraction of records with fields to be quantified.

Once the expression is constructed, click on  to send the search expression to the statistics form.

You must choose a single method for the extraction of records: By Mfn or By searching.

Supply the data in the form, and click on Send to obtain the corresponding results

 

Generating the table

Using the values supplied in the statistics form a frequency matrix is built or a contingency is requested.

Conjuntamente con el despliegue del cuadro se despliegan las  In conjunction with the table, options to send the results to one of the following are displayed:

  • A spreadsheet (.xls)
  • A document (.doc)
  • Generate an animated graphic
  • Generate a graphic without animation

A spreadsheet

A table is constructed which opens in the associated program of the client, with .xls files

Document

A table is constructed which opens in the associated program of the client, with .doc files

Animated graphic

Results are sent to a plug-in generator of animated graphics

Graphic without animation

Results are sent to a plug-in generator of graphics without animation
