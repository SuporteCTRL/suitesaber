Generating a new database from a copy of the defined resources from another database

This option generates the resources required to define a new database, from the already defined resources of an existing database.

The procedure runs through the folders of the selected database using this as a model and copying them into a folder with the name of the new database.

It creates the file .par for the new database and includes this in the file bases.dat

Once the process is finalised, the system Administrator must be assigned.
