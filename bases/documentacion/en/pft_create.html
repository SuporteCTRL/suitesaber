Creation of display format

The list of existing fields is showed according to the Fields definition table.

In the left window select the fields to use in the display format and pass them to the right window by using the directions arrows. Use the links up and down to modify the fields order in the generated format.

Once the selection is done, indicate if you want to generate the export in a table (separating the data in rows and columns) or in a paragraph form (separating the data with a line). The generated format will be shown in the window.

Click on the link Create the database to create the database
