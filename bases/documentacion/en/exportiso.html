Exporting the database to an ISO or TXT file

The  ISO files or the TXT files that are created through the export feature are stored in the folder ./bases/wrk. To export a database and obtain an export file select the records that you want to export through the available options:

  • Mfn range

  • Search

  • A letter of the alphabet

Once you have selected the records to export, insert the name of the ISO or TXT file that you will create into the box, and click on.

When the file is created one page with the results and a link to the file will appear. If you want to download the file, right click on the link to see the option that will allow you to do so.

Afterwards, while importing, the file you obtained in the export process will be shown as part of the list of files stored in the folder ./bases/wrk.

Important Note: If the database has many records, it is advisable to use the CISIS utility to create the ISO or TXT files.
