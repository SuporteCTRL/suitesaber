Help files for the fields of the databases

To define the help-pages for the fields in the data-entry form, follow the following steps:

  1. Select from the menu Field Definition Table (FDT) the field for which you will create a help page 
  2. After selection the properties of the field will be shown as defined in the Field Definition Table (FDT)
  3. Click on Preview to present the help-page, if already available
  4. Click on Edit help-page to create or edit the existing page. The HTML editor with which you can write the text of the help-page will be shown.
  5. When ready with the editing of the help-page, click on Save page to store the page in the help-system.
  6. The help-pages for the fields are stored in /bases/dbn/ayudas/tag_xxx.html, where dbn is the name of the active database and xxx  represents the field tag for which the help-page is created. This name will be passed to the system automatically and should not be modified since it will act as a reference at data entry to provide a help-page.
  7. When the help-pages are ready, edit the FDT of the database to indicate for which fields, in the corresponding column, help-pages have been defined.
