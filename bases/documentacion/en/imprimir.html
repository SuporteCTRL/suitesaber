This procedure allows you to generate an output format - or use a previously generated format - to send information directly to a word processor.

The records can be selected in two ways

  1. Through a range of records (from - up to)
  2. Applying a search expression to the database. To do this click on the search icon and construct the search expression

Once the records are selected, choose a existing format or build a new format.

In order to select an existing format, consult the list that is displayed under Selection of formats already defined: click on the format you wish to use. Then click on the icon which will present a window with the selected format. If you know the formatting language Cds/isis, you can edit the format to make changes or adjustments; otherwise, do not modify the format as it will generate errors and you will obtain no results when requesting a listing.

Click on Send in order to send the format to the output file. You can test the format by selecting the records either by MFN status or performing a search on the database. Once the selection has been carried out click on  Send to Word (in order to send the test to a word processor)  or on  Preview to see the outcome of the format through a screen display.

In order to create a new format select the fields you want to include in the output from the list. To select the fields double click on the name or highlight with the help of the "mouse" and click on the corresponding arrows. You can use the keys [Ctrl] and [Shift] to select the fields. The arrows simply transfer the selected fields from one window to another and the double arrows transfer all the fields.

You can use the buttons up or down to change the position of the selected fields.

When you have finished with your selection click on Generate format and the generated format will be displayed in the window. If you know the formatting language Cds/isis, you can edit the format to make changes or adjustments; otherwise, do not modify the format as it will generate errors and you will obtain no results when requesting a listing.

You can test the format by selecting the records either by MFN status or performing a search on the database. Once the selection has been carried out click on Send to Word (in order to send the test to a word processor)  or on Preview to see the outcome of the format through a screen display.

To save the format you have developed provide a name (using letters and numbers only without spaces) and provide a description with which to present this format in the list of defined formats.
