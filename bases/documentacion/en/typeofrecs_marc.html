Record Types (MARC format)

This option helps to define the record types which will be managed by the database when the system uses the MARC format.

The record type is found in field 3006 of the Leader and assists in identifying the format of the fixed field (008) to be used in the bibliographic description of the document.

Defining the record type requires 2 components:

  1. Identify the field (tag) of the database from which the record type corresponding to the document is going to be obtained (or to be entered in the case of a new record). The current version of ABCD allows the definition of up to two fields whose values determine the format of the fixed field (008) to be present in the data entry form.
  2. Describe the record types with the following parameters:
    • Name of the field definition table (FDT) which holds the elements used to construct the data entry worksheet to capture the data
    • Value of the fields identified in (1) which are regarded as belonging to that record type
    • Description of the record type which will be displayed to the user at the time of data entry which precedes the selection of the corresponding type

To maintain consistency in the definitions of ABCD, the fixed fields are defined by a FDT that contains the description of the data elements associated with the record type and with all the characteristics defined in the process of creating the FDT of the database. To edit the FDT associated with the fixed field for a record type, click on the link Edit. This will open a new window where the characteristics of the data elements that comprise the fixed field are given.
