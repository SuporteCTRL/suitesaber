Exportação da base de dados para um arquivo TXT com rótulos

Para exportar uma base de dados na forma de um arquivo TXT com rótulos, se utiliza uma das tabelas de conversão já definidas na opção de Importação. (Ver Tabelas de conversão)

Para executar a exportação, siga o seguinte procedimento:

  1. Selecione a tabela de conversão a utilizar no processo. Para isso clique na opção Selecionar que acompanha cada tabela de conversão exibida na janela
  2. Selecione os registros que deseja exportar através das opções previstas:
    • Faixa de Mfn
    • Busca
    • Uma letra do glossário
  3. Selecione a saída onde quer ver o arquivo TXT com os registros exportados, segundo as seguintes opções:
  4. Processador de texto
  5. Vista prévia

Uma vez selecionados os registros a exportar, digite na caixa o nome do arquivo ISO a gerar e clique no ícone .

Ao terminar a geração do arquivo ISO é exibida uma página com o resultado e um link para o arquivo ISO gerado. Si deseja baixar o arquivo ISO, posicione o mouse sobre o link e clicando com o botão direito terá acesso a opção que permite salvar o ISO no seu computador.

Posteriormente, no processo de importação, se exibirá o ISO gerado como parte da lista dos arquivos residentes na pasta ./bases/wrk.

Nota importante.: Se a base de dados tem muitos registros, recomenda-se usar o utilitário MX para a geração do arquivo ISO
