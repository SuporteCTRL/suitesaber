Criação de novos formatos de exibição

Este procedimento permite gerar um formato de saída - ou modificar um previamente criado. Os formatos obtidos podem ser usados para exibir informações na tela ou enviar informações diretamente para um processador de texto ou a um processador de planilhas.

Para selecionar um formato já existente, veja a lista abaixo Formatos: e dê um clique sobre o formato que deseja utilizar. Em seguida, clique em Editar que irá apresentar uma janela com formato selecionado. Se conhece as regras para elaboração de formatos do Cds/isis, pode editar o formato para fazer alterações ou adaptações; se não conhece, não modifique a saída já que se gerar erros não terá nenhum resultado ao ordenar a impressão do relatório.

 Na janela que exibe o formato, uma vez realizadas as alterações, clique em enviar para mandar o formato para a tela de geração da saída.

Para criar um novo formato clique sobre a seção Criar um formato e selecione da lista de campos aqueles que deseja incluir na saída. Para selecionar os campos dê um clique duplo sobre o nome ou destaque-os com a ajuda do "mouse" e clique sobre as setas correspondentes. Pode usar as teclas [Ctrl] e [Shift] para selecionar os campos. As setas simples transferem os campos selecionados de uma janela a outra e as setas duplas transferem todos os campos.

Pode usar os botões subir ou descer para mudar a posição dos campos selecionados.

Ao terminar a seleção clique sobre uma das opções indicadas em Criar formato e aparecerá na tela o formato criado. Se conhece a linguagem de formatação do Cds/isis, pode editar o formato para realizar alterações ou modificações; se não conhece, não modifique a saída já que se gerar erros não terá nenhum resultado ao ordenar a impressão do relatório.

Uma vez finalizada a seleção/edição/criação de um formato pode ir para Criar a saída correspondente

Os registros podem ser selecionados de duas formas

  1. Através de uma faixa de registros (de - até)
  2. Executando uma expressão de busca na base de dados. Para isso clique no ícone de busca e construa a expressão correspondente

 Uma vez realizada a seleção clique em Enviar a um processador de textos (para enviar a faixa de registros a um processador de textos), Enviar a un processador de planilhas,    ou sobre Preview para ver o resultado do formato na tela.

Para imprimir a saída clique em Criar saída e, em seguida, colocar a informação de acordo com o método a ser utilizado para a seleção dos registros,  seja por faixa de Mfn ou realizando uma busca na base de dados.

Uma vez realizada a seleção clique em Enviar a um processador de textos  (para enviar a faixa de registros a um processador de textos), Enviar a un processador de planilhas  , ou sobre Preview para ver o resultado do formato na tela.

Para salvar o formato clique em Salvar formato  e dê um nome (usando letras e números somente e sem espaços em branco) e dê uma descrição breve na qual se explica este formato na lista de formatos já definidos.
