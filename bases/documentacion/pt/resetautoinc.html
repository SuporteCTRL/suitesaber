Restaurar o número de inventário ou o número de controle de umna base de dados

Número de inventário

O número de inventário é uma numeração, que pode ser automática no ABCD, para a base de dados, para os exemplares (copies). Este número está definido na FDT da base de dados com o tipo de campo autoincrement e se incrementa em cada registro criado, a partir do último valor armazenado.

O último valor armazenado é obtido de um archivo denominado control_number.cn localizado na pasta data da base de dados em questão.

Esta opção modifica o valor do último número atribuído com o objetivo de restabelecer a numeração dos registros.

Número de controle

O número de controle é um campo obrigatório nas bases de dados bibliográficas cujos exemplares irão ser manipulados através da base de dados copies, por ser o campo de relação entre o objeto e suas cópias.

Como no caso anterior, ABCD pode encarregar-se de gerar de forma automática este número de controle através de uma numeração sequencial. Como no caso anterior, o último número atribuído é mantidono arquivo control_number.cn localizado na pasta data da base de dados em questão.


Esta opção modifica o valor do último número atribuído com o objetivo de restabelecer o valor do número de controle dos registros.
