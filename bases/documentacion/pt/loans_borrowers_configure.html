Configurar a base de dados de usuários.

Esta opção permite configurar no sistema a base de dados de usuários que poderão usar o serviço de empréstimo.

Os parâmetros requeridos são:

Prefixo para obter o número de usuário Especifica o prefixo que é colocado antes do número do usuário para localizar a informação na base de dados de usuários. Fica armazenado na pasta def da base de dados de usuários (users) sob o nome loans_uskey.tab
PFT para extrair o número de usuário Depois de localizado o registro do usuário, é aplicado este formato de extração para obter o número do usuário. Fica armazenado na pasta def da base de dados de usuários (users) sob o nome loans_uskey.pft
PFT para extrair o tipo de usuário A aplicação das políticas de empréstimo depende do tipo de usuário e do tipo de item a emprestar. Este formato de extração permite extrair o tipo de usuário que fica armazenado na pasta def da base de dados de usuários (users) sob o nome loans_ustype.tab
PFT para extrair a vigência do usuário

Através deste formato é definida a vigência do usuário. Deve incluir uma mensagem e um código, separados pelo caracter |.
Os códigos definidos são:

0=Usuário ativo
1=Usuário inativo

Fica armazenado na pasta def da base de dados de usuários (users) sob o nome loans_usvig.tab
PFT para exibir a informação do usuário Este formato de extração apresenta a informação do usuário para a emissão de status da conta e outros relatórios relacionados com a gestão de empréstimos. Fica armazenado na pasta def da base de dados de usuários (users) sob o nome loans_usdisp.tab
