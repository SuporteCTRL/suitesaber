Criar uma base de dados copiando a estrutura já existente em WinIsis

Pode-se criar uma base de dados carregando a FDT, FST e PFT existentes numa definição local do WinIsis.

Para isso o ABCD permite carregar a FDT a partir da máquina local convertendo-a na estrutura da FDT mantida neste sistema.

Igualmente permite carregar a FST e um formato de saída (PFT).

Note que o formato de saída selecionado deve ser escrito usando pré e pós literais que reproduzem corretamente uma página HTML. Formatos de exibição que contêm os elementos da linguagem de formatação definidos para a visualização em WinIsis e não para um navegador (browser), produzirão erro de sintaxe ao serem usados na visualização no ABCD.

Uma vez completadas as três etapas o ABDC criará a base de dados e a adicionará à lista de bases de dados existentes.

Para que a base de dados possa ser operada por outros usuários que não tenham as permissões de “Administrador do Sistema”, é necessário entrar na opção de Administração de usuários, criar ou selecionar os usuários que podem acessar a base e atribuir as permissões adequadas.
