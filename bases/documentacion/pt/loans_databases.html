Bases de dados de itens disponíveis para  empréstimo

Por meio deste procedimento podem ser selecionadas bases de dados definidas no ABCD que contém itens disponíveis para  empréstimo  

Selecione da lista apresentada uma base de dados e depois clique em Ir para definir os parâmetros e formatos requeridos para fazer a ligação entre a base de dados e o sistema de empréstimo.

Em linhas gerais, é necessário definir:

  • A chave de acesso aos títulos
  • Como determinar o total de exemplares do título
  • Como extrair a informação de cada exemplar
  • Como apresentar a informação catalográfica do título
  • A informação do título a ser transferido para a base de dados de empréstimos, para identificação dos materiais emprestados e para obter as estatísticas
  • Como apresentar a informação do título emprestado, depois de estar armazenado na base de dados de empréstimos

Uma vez selecionada a base de dados, será apresentado um formulário de entrada, onde são solicitados os parâmetros e formatos requeridos pelo processo.
