Solicitação de Licitações

Neste processo são mostradas tanto as sugestões que são aprovadas como aquelas que estão em processo de licitação. A fórmula de busca que se aplica sobre a base de dados é : STA_1 or STA_3, a qual mostra os registros cujo status seja 1=aprovado ou 3=em licitação

Mostra as seguintes informações:

  • Número sequencial atribuído de forma automática à sugestão
  • Data de aprovação da sugestão
  • Número de copias (exemplares) aprovadas
  • Título do objeto
  • Data de início do processo de licitação
  • Fornecedores selecionados:
    • Nome do fornecedor
    • Número de exemplares solicitados
    • Montante pago
    • Observações
    • Selecionado (Sim/Não)
    • Data da seleção
    • Data ISO da seleção
    • Número da nota emitida

Os registros obtidos podem ser classificados segundo três critérios,e para cada critério existe um formato (PFT) que permite apresentar a informação na ordem selecionada e fornecer as etiquetas das colunas da tabela:

Você pode enviar esta saída para um editor de texto, planilha eletrônica, utilizando os links que são apresentados no menu Ação,localizado no topo. Neste caso, a saída preparada exibe uma lista para cada fornecedor selecionado no processo de listagem de um objeto; isto é, se um objeto possuí três fornecedores o objeto apresentará três linhas na tabela. Isso facilitará o agrupamento das solicitações com base no fornecedor.
