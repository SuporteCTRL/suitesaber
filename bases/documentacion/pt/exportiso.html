Exportação da base de dados para um arquivo ISO ou para um arquivo TXT

Os arquivos ISO ou os arquivos TXT gerados na exportação são armazenados na pasta ./bases/wrk. Para exportar uma base de dados e gerar um arquivo de exportação selecione os registros que deseja exportar através das opções previstas:

  • Faixa de Mfn
  • Busca
  • Uma letra do glossário

Uma vez selecionados os registros a exportar, digite na caixa o nome arquivo ISO ou TXT que será gerado e clique em .

No fim da geração do arquivo é exibida uma página com o resultado e um link para o arquivo. Se desejar baixar o arquivo, posione o mouse sobre o link e clique com o botão direito e terá acesso a opção que permitirá gravar o arquivo no seu computador

Posteriormente, no processo de importação, se mostrará o arquivo obtido na exportação,  como parte da lista dos arquivos existentes na pasta ./bases/wrk.

Nota importante.: Se a base de dados tem muitos registros, recomenda-se usar os utilitários do CISIS  para a geração do arquivo ISO ou TXT
