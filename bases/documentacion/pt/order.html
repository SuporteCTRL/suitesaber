Elaboração da ordem de Compra

Nesta lista são mostrados os fornecedores que foram aprovados para executar a compra dos objetos para os quais ainda não foi emitida a ordem decompra.

Se uma sugestão para a compra foi aprovada para mais de umfornecedor, são mostrados em linhas separadas os fornecedores selecionados. Isto é feito para facilitar a elaboração da ordem de compra.

Para saber se, para um fornecedor cuja cotação foi aprovada, foi emitida a ordem de compra, ao formatar a requisição é utilizado o comando npost da linguagem deformatação para localizar na base de dados de Ordens de Compra  (purchaseorder) a chave composta pelo número de requisição mais o número de licitação, precedido pelo prefixo RNBN.:

if npost([`purchaseorder`],`RNBN_`,mpu,s(v1[1]),`_`v300^r)<>0 then `OK` fi

Então, se ao formatar a requisição aparecer no formato o termo OK,  não será mostrada a linha correspondente à cotação aprovada de determinado  fornecedor

A informação apresentada é a seguinte:

  • Número sequencial da sugestão
  • Referencia do objeto
  • Fornecedor aprovado
  • Data de aprovação da compra
  • Número de copias/exemplares aprovados
  • Preço de cada exemplar

Clique em para selecionar os itens que serão incluídos na ordem de compra em processo de elaboração.

Uma vez realizada sua seleção, clique sobre Ordem de compra.

Para a seleção dos registros é aplicada a fórmula de busca STA_3, que  corresponde à requisição da etapa de Seleção de fornecedores

A informação recuperada pode ser ordenada de 4 formas, e para cada forma existe um formato pré-definido:
