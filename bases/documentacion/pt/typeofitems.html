Tipos de material

Esta tabela descreve as políticas a aplicar para os diferentes tipos de material existentes na base de dados. O tipo de material  referenciado na tabela é gerado pelo formato de extracção que se aplica à base de dados com os materiais que são objeto de empréstimo; este formato, entre outros dados, deverá incluir o tipo de material a incluir na tabela para ser possível estabelecer a respectiva política.

Se o formato de extração não criar nenhum resultado então o operador do empréstimo deve decidir o tipo de empréstimo a aplicar seleccionando uma das opções da tabela Tipos de Empréstimo.

Quando se grava o registo de empréstimo, é guardado o tipo de material que foi atribuido para efeito de aplicação das políticas de devolução/multa/inibição.

Descrição da tabela

Campo Descrição
Tipo de material Valor que devolve o formato de extração aplicado à base de dados documental.
Descrição Denominação do tipo de material
Tipo de utilizador Seleccione o tipo de utilizador para a qual se aplica a política do tipo de material. Se não for seleccionado nenhum utilizador, a política será aplicada a todos os utilizadores
Duração do empréstimo Número de dias/horas para o cálculo da data de devolução
Unidade Unidade de medida da duração do empréstimo (Dias/Horas)
No. de renovações Número de renovações permitidas
Multa por dia de atraso Montante da multa a cobrar por cada dia/hora de atraso na devolução do empréstimo 
Dias de suspensão por dia de atraso Indica quantos dias se deve inibir o utilizador por cada dia/hora de atraso na devolução
Em reserva. Multa por dia de atraso Se o material está reservado, especifica a multa a aplicar por cada dia de atraso na devolução
Em reserva. Suspensão por dia de atraso Se o material está reservado, especifica os dias de inibição que se aplicam por cada dia de atraso na devolução
