Criar nova sugestões

Antes de inserir uma nova sugestão na base de dados, deve-se verificar se trata da adquisição de um novo objeto ou de exemplares adicionais a um objeto já existente.  Neste caso deve submeter a base de dados onde se encontra a informação catalográfica do objeto e o Número de identificação do objeto, tal como foi designada na base de dados correspondente.

Se tratar de um novo objeto, tambén pode inserir o nome da base de dados onde se catalogará o objeto.

Os dados obrigatórios para adicionar são :

Etiqueta  
2 Estado da sugestão
3 Tipo de aquisição
5 Base de dados onde se cataloga o objeto
6 Número de identificação do objeto (Obrigatorio se o tipo de aquisição selecionado  for Exemplares adicionais )
16/17 Se deve indicar o autor pessoal ou o autor corporativo
18 Título
200 Nome da pessoa/departamente que sugeriu a compra do objeto
210 Data da sugetão
211 data ISO da sugestão
220 Número de exemplares solicitados

A informação inserida se armazena na base de dados suggestions e novos registros são automaticamente atribuído um número sequencial que é utilizado como um elo no processo de monitoramento do estado da sugestão..

Para a inserção do número sequencial, se mantém na pasta data da base de dados suggestions um arquivo denominado control_number.cn noqual posa se gravado o último número inserido.
