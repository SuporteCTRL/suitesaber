Capturar registros de outras bases de dados

Selecione a base de dados da qual irá capturar o registro. Use a lista de seleção que se encontra na caixa Capturando da base de dados:. A base selecionada permanecerá ativa até que mude para outra base de dados.

Uma vez selecionada a base de dados, na caixa da direita será exibida a lista alfabética da entrada principal dos registros da base de dados correspondente, a qual deve se especificarse na Tabela de Definição de Campos . Clique numa entrada para visualizar seus detalhes.

Para mover-se na lista de registros, use o Índice alfabético ou clique em Mais têrmos para continuar visualizando a partir do último têrmo exibido na janela. Se deseja ir a um título específico, digite as primeiras letras ou palavras do mesmo na janela Ir para e pressione [Enter] ou clique em continuar.

Pode usar a borda do quadro para ampliar ou reduzir a largura das janelas.

Se quiser copiar na base ativa o registro exibido, clique em para ativar o modo de entrada. Pode editar os campos antes de copiá-los, e as mudanças não serão refletidas no registro original.

Não feche a janela se vai capturar mais de um registro.

A última base da qual se capturou registros, permanecerá ativa durante os processos seguintes de captura.
