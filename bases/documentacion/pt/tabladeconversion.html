Tabela de conversão para exportação da base de dados para um arquivo TXT com etiquetas

A tabela de conversão é usada tanto para exportar com para importar registros, aplicando-se em cada caso as seguintes regras:

Use a caixa  Delimitada com tabuladores, para importar uma planilha Excel salva no formato TXT. Neste caso a coluna Etiqueta no arquivo TXT deverá conter o número da coluna (começando por 1), onde se encontra o campo correspondente na planilha Excel exportada.

Consta das seguintes colunas:
Etiqueta no arquivo TXT Etiqueta com a qual se identifica o campo no arquivo TXT de entrada ou de saída. As etiquetas devem começar com $$ e terminar com :
Tag Etiqueta (Tag) do campo na base de dados.
Subcampos Delimitadores de subcampos. Se o primeiro subcampo não tiver delimitador, deixar um espaço em branco.
Substituir por Se um campo tiver subcampos, indicar aqui os caracteres que irão substituir os subcampos do campo. O primeiro caractere desta lista irá substituir o primeiro subcampo da lista de subcampos. Se o primeiro subcampo não será substituído por nenhum delimitador, deixar um espaço em branco.
Separador de ocorrências Indique aqui o caractere com o qual são separadas as ocorrências de um campo repetitivo.
Formato de extração Este formato é aplicado ao registro para gerar a saída do campo associado. A aplicação do formato de extração só é executada no processo de exportação. Se forem submetidos delimitadores de subcampo e o formato de extração, este último é aplicado no processo (não implementado).
