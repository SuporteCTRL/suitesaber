Parâmetros necessários para configurar a base de dados de objetos para empréstimo

1. Prefixo para localizar o Número de Registro

Este prefixo é acrescentado ao número de registro/código de barras fornecido pelo operador de empréstimos a fim de localizar o titulo correspondente.  É armazenado na pasta def da base de dados selecionada, com o nome de loans_conf.tab
2. Prefixo para localizar o número de classificação Este prefixo é acrescentado ao número de classificação informado pelo operador de empréstimo, a fim de localizar o titulo correspondente.  É armazenado na pasta def da base de dados selecionada e no mesmo arquivo loans_conf.tab 
3. PFT para calcular o total de exemplares do título Uma vez localizado o item objeto de empréstimo, é aplicado este formato sobre o registro localizado por um dos dois métodos anteriores para obter o total de exemplares do título. É armazenado na pasta def da base de dados selecionada, com o nome loans_totalitems.pft
4. PFT para extrair o Número de Registro Este formato é aplicado sobre o item objeto de empréstimo a fim de extrair os números de registro do título e identificar o exemplar solicitado. É armazenado na pasta def da base de daos selecionada, com o nome loans_inventorynumber.pft
5. PFT para extrair o número de classificação
6. PFT para exibir o registro objeto de empréstimo Este formato é aplicado sobre o registro localizado na base de dados de objetos para empréstimo para efeitos de visualizar o registro. É armazenado na pasta def da base de dados selecionada, com o nome loans_display.pft
7.  PFT para obter o tipo de registro da base de dados de objetos para empréstimo Este formato é aplicado sobre o registro localizado na base de dados de objetos para empréstimo para obter o tipo de registro do exemplar solicitado. Este tipo de registro é combinado com o tipo de usuário para determinar a política de empréstimo do exemplar solicitado.É armazenado na pasta def da base de dados selecionada, com o nome de loans_typeofobject.pft
8. PFT para guardar o registro na base de dados de empréstimos Este formato é aplicado sobre o registro localizado na base de dados de empréstimos e é utilizado para guardar a informação catalográfica do exemplar emprestado ao usuário. É armazenado na pasta def da base de dados selecionada, com o nome de loans_store.pft
9. PFT a ser utilizada para exibir o objeto a partir da base de dados de empréstimos Este formato é utilizado poder exibir a informação dos empréstimos pendentes a partir da base de dados de empréstimos. É armazenado na pasta def da base de dados selecionada, com o nome de loans_show.pft

Pode-se usar o vínculo Testar e informar um MFN para aplicar os formatos fornecidos sobre a base de dados de objetos para empréstimo.

Para salvar os formatos, dê um clique sobre Atualizar.
