Arquivo dbn.par

O arquivo dbn.par é um recurso próprio da estrutura CDS/ISIS que indica a localização dos recursos de cada base de dados.

Na atual versão do ABCD este arquivo contém:

  • O caminho para o diretório onde se encontra a base de dados
  • O caminho para o formato de início no processo de atualização de cada registro
  • O caminho para o formato do final do processo de atualização de cada registro

Exemplo:

biblo.*=%path_database%biblo/data/biblo.*
prologoact.pft=%path_database%www/prologoact.pft
epilogoact.pft=%path_database%www/epilogoact.pft

Os formatos de visualização não estão incluídos pois o ABCD gera o caminho a partir dos parâmetros de configuração.

O nome %path_database% é substituído no arquivo dbn.par pelo caminho atual do separador bases, tal como se especifica na variável $db_path do script config.php. Este, permite usar o diretório bases sem qualquer impacto nos caminhos dos recursos das bases de dados.

Se pretender colocar os arquivos da base de dados fora do diretório definido por default, altere o arquivo dbn.par para indicar o caminho absoluto.

Se um formato incluir uma referência a outro formato ou se usa o comando REF para relacionar bases de dados, tanto os formatos que são chamados como as bases de dados relacionadas devem ser indicadas no correspondente arquivo dbn.par
