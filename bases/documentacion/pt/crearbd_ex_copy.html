Gerar uma nova base de dados a partir da cópia de recursos definidos para outra base

Esta opção gera os recursos necessários para definir uma nova base de dados a partir dos recursos já definidos para uma outra base de dados existente.

O procedimento percorre as pastas da base de dados selecionada como um modelo e copia os recursos para uma pasta com o nome da nova base de dados.

É criado o arquivo .par para a nova base de dados e o nome da base é incluído no arquivo bases.dat.

Depois de terminado o processo de criação, deve-se definir os operadores que podem gerir a nova base de dados, porque do contrário só o Administrador do sistema terá acesso a ela.
