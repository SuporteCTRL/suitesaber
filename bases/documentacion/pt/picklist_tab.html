Captura de termos normalizados através de uma tabela

Esta opção do ABCD permite criar tabelas de valores que serão exibidos na entrada de dados correspondente. A forma como se exibe a tabela dependerá do Tipo de entrada assignada ao campo na FDT (checkbox, option, select single, select multiple). As tabelas de valores são armazenadas na pasta def da base de dados selecionada.

Para criar uma nova tabela coloque no quadro Nome o nome da mesma, com a extensão .tab e clique sobre editar/criar.
Para visualizar uma tabela existente clique na imagem ou no nome da tabela correspondente.

Para editar uma tabela existente clique no nome da mesma e clique em editar/criar.

Para atribuir um campo a uma tabela existente, clique no nome da mesma e clique em atualizar FDT.

Para eliminar uma tabela existente clique no nome da mesma e clique em excluir. A tabela será deletada da pasta, mas permanecerá associada ao campo respectivo na FDT. Lembre-se, então, de excluir a entrada na FDT.

A lista de tabelas disponíveis exibe todos os arquivos com a extensão .tab localizados na pasta def da base de dados selecionada.

Criação/Edição de tabelas

Uma vez que o nome do arquivo tenha sido digitado na caixa Nome, clique em criar/editar. A seleção será exibida numa tabela con duas colunas: Código |  Termo

Se somente a coluna termo é preenchida, no procedimento de entrada serão exibidos os valores colocados nesta coluna, os quais serão transferidos ao campo correspondente utilizando o método de seleção apropiado ao tipo de entrada do campo definido na FDT.

Se deseja transferir tanto o código como o termo associado, então a estrutura do campo que vai reciber o valor da tabela deve contemplar 2 subcampos. No primeiro subcampo, será incluído o valor correspondente ao código e no segundo subcampo se colocará o valor correspondente ao têrmo.No procedimento de entrada serão exibidos somente os valores presentes na coluna termo, mas se armazenará o código mais o termo utilizando-se os subcampos correspondentes.

Ao terminar a edição da tabela, clique no botão Atualizar para salvar a tabela criada/modificada e assigná-la ao campo correspondente na FDT.

Tip: Para utilizar numa base de dados A uma tabela já definida para a base de dados B, preceda o nome da Picklist com o caminho correto onde está localizada a tabela que deseja usar.
