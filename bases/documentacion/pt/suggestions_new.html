Criar nova sugestões

Antes de inserir uma nova sugestão na base de dados, deve-se verificar se trata da aquisição de um novo objeto ou de exemplares adicionais a um objeto já existente.  Neste caso deve informar a base de dados onde se encontra a informação catalográficado objeto e o Número de identificação do objeto, tal como foi atribuído na base de dados correspondente.

No caso de se tratar de um novo objeto, também deve inserir o nome da base de dados onde será catalogado o objeto.

Os dados obrigatórios para adicionar são :

Etiqueta  
2 Estado da sugestão
3 Tipo de aquisição
5 Base de dados onde será catalogado o objeto
6 Número de identificação do objeto (Obrigatorio se o tipo de aquisição selecionado  for Exemplares adicionais)
16/17 Deve indicar o autor pessoal ou o autor corporativo
18 Título
200 Nome da pessoa/departamento que sugeriu a compra do objeto
210 Data da sugestão
211 data ISO da sugestão
220 Número de exemplares solicitados

A informação inserida é armazenada na base de dados suggestions e aos novos registros é atribuído automaticamente  um número sequencial que é utilizado como um elo no processo de monitoramento do estado da sugestão..

Para a inserção do número sequencial, é mantido na pasta data da base de dados suggestions um arquivo denominado control_number.cn no qual é gravado o último número inserido.
