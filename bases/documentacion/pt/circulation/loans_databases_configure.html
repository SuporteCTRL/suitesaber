Parâmetros requeridos para configurar a base de dados com os itens destinados ao empréstimo

Parâmetro
Exemplo
 
 

Pft para exibir a informação bibliográfica do objeto (loans_display.pft)
V50^a,” “v245^a,” “d260,v260^a, …

Pft para armazenar a informação do título na base de dados de transações (para visualização e estatísticas)

(loans_store.pft)
 `^a`,v250^a,’^b’,v245^a,’^c’v260^c ….

   O objetivo deste formato é construir um campo com subcampos que contenha a informação bibliográfica que será armazenada como parte da transação do empréstimo. O resultado produzido pelo formato é armazenado no campo 100 do arquivo de transações de empréstimo (trans)

Pft para extrair a informação da base de dados de transações (loans_show.pft)
Mhl, v100,

Por implementar:

Si el registro bibliográfico contiene las existencias, esto es, no se trabaja con loanobject, se requieren adicionalmente los siguientes parámetros:

 
 

Prefixo a utilizar para localizar um número de inventário (loans_conf.tab)
NI=

Prefixo a utilizar para localizar un número de classificação (loans_conf.tab)
SN_

Pft para extrair o número de classificação (loans_conf_tab)
V50^a

Tag do campo com as cópias (loans_conf.tab)
900

Formato para extrair o tipo de material

If iocc=1 then ‘R’ else ‘L’ fi

O resultado deste formato deVe ser um dos tipos de objeto considerados na política de empréstimos

Formato para extrair o número de inventário
 
Formato para extrair o volume
 
Formato para extrair a cópia
