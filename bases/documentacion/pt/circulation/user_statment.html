Para solicitar um estado de conta, insira o código do usuário e clique em Buscar. Pode utilizar o botão Listar para obter uma lista alfabética dos usuários registrados.

Se não souber o código del usuário, pode inserir o número de inventário de algum objeto que esteja em poder do mesmo, utilizando para isso a caixa No. de inventário.  Se clicar em Listar serão exibidos os números de inventário dos objetos que atualmente estão em poder de algum usuário.
