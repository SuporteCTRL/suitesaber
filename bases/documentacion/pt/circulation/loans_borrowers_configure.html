Configurar a base de dados de usuários.

Esta opção permite configurar no sistema, a base de dados com os usuários que têm acesso ao serviço de empréstimo.

Os parâmetros requeridos são:

Prefixo para obter o número de usuário Especifica o prefixo que se coloca antes do número do usuário introduzido para localizar a informação da base de dados de usuários. Fica armazenado no directório def da base de dados de usuários sob o nome loans_uskey.tab
PFT para extrair o número de usuário Depois de localizado o registo do usuário, aplica-se este formato de extração para obter o número do usuário. Fica armazenado no diretório def da base de dados de usuários sob o nome loans_uskey.pft
PFT para extrair o tipo de usuário A aplicação das políticas de empréstimo depende do tipo de usuário e do tipo de item a emprestar. Este formato de extração permite extrair o tipo de usuário. Fica armazenado no directório def da base de dados de usuários sob o nome loans_ustype.tab
PFT para extrair a vigência do usuário

Através deste formato define-se a vigência do usuário. Deve incluir uma mensagem e um código, separados ambos pelo caracter |.
Os códigos definidos são:

0=Usuário ativo
1=Usuário inativo

Fica armazenado no directório def da base de dados de usuários sob o nome loans_usvig.tab
PFT para visualizar a informação do usuário Este formato de extração apresenta a informação do usuário para a emissão de pontos de situação da conta e outros relatórios relacionados com a gestão dos empréstimos. Fica armazenado no diretório def da base de dados de usuários sob o nome loans_usdisp.tab
