Pesquisa Avançada 

O formulário de pesquisa avançada estrutura-se na secção de Definição de Bases de Dados e contém os campos que estão indexados na FST e que são pesquisáveis a partir do índice de pesquisa.

Para fazer uma pesquisa avançada execute os seguintes passos:

  1. Seleccione o campo objecto da sua pesquisa
  2. Clique em Índice para consultar o Índice de pesquisa do campo
  3. Seleccione no índice de pesquisa os termos objecto da sua consulta. Para isso tem duas opções:
    • Seleccione o termo e clique em Pesquisar para fazer a pesquisa directamente sobre o termo seleccionado
    • Seleccione o termo e clique sobre Transferir o termo seleccionado para transferir o termo para o formulário de pesquisa e construir uma expressão de pesquisa que combine informação proveniente de mais de um campo
  4. Se não quiser utilizar o índice de pesquisa, escreva na caixa correspondente palavras relevantes da informação que deseja localizar. Para que esta opção funcione, é necessário indexar o campo com a técnica 4 (por palavras) na FST. Se não seleccionar nenhum campo da lista de selecção de campos pesquisáveis, a pesquisa incidirá sobre todos os campos da base de dados que tenham sido indexados.
  5. Logo que a sua selecção esteja completada, clique sobre para fazer a pesquisa na base de dados e localizar os registos, ou sobre para limpar os termos de pesquisa.

Os registos localizados apresentam-se na janela da base de dados para  facilitar a edição.
