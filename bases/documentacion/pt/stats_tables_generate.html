Antes de usar esta opção, é necessário configurar as variáveis da base de dados envolvidas nas estatísticas e opcionalmente, nas tabelas de frequência e/ou contingência, onde estas se aplicam.

Para configurar as variáveis envolvidas nos processos de estatística, use a opção configuração de estatísticas ou Modificar definição de bases de dados do menu inicial do módulo de administração e execute

Geração de estatísticas

Usar uma tabela existente

Se houver tabelas, é exibido um menu tipo pulldown com as tabelas previamente definidas na opção Modificar definição de bases de dados. Selecione a tabela que deseja utilizar.

Criar uma tabela

é exibida a lista de campos da base de dados que tenham sido selecionados para a obtenção de estatísticas. Selecione o campo a utilizar nas linhas e opcionalmente, o campo a utilizar nas colunas.

Obs: coloque como linhas o campo que possui a maior diversidade de valores.

Gerar saída

Nesta seção indique o método de seleção dos registros para os quais deseja obter uma matriz de frequência ou de contingência.

  • Por faixa de Mfn
  • Por pesquisa

Se selecionar por pesquisa será exibida, numa janela pop-up, o formulário de pesquisa avançada para ajudar na construção da expressão de pesquisa a ser aplicada á base de dados, para extrair os registros cujos campos serão quantificados.

Uma vez construída a expressão, clique em   para enviar  a expressão de pesquisa ao formulário de estatísticas.

Deve escolher somente um método de extração de registros: por Mfn ou por pesquisa.

Uma vez fornecidos os dados no formulário, clique em Enviar para obter os resultados correspondentes.

 

 

 

Geração da tabela

Com os valores fornecidos no formulário de estatísticas, cria-se a matriz de frequência ou contingência solicitada.

Junto com o desdobramento do quadro, são desdobradas as opções para enviar o resultado por um dos seguintes meios:

  • Uma planilha eletrônica (.xls)
  • Um documento (.doc)
  • Gerar um gráfico animado
  • Gerar um gráfico não animado

Planilha eletrônica

A tabela de estatísticas é construída e é aberto o programa associado, no cliente, aos arquivos .xls.

Documento

A tabela de estatísticas é construída e é aberto o programa associado, no cliente, aos arquivos .doc.

Gráfico animado

Os resultados são enviados a um plug-in gerador de gráficos animados

Gráfico não animado

Os resultados são enviados a um plug-in gerador de gráficos não animados.
