Criação de uma nova sugestão

As sugestões inseridas na base de dados são numeradas sequencialmente afim de criar relações com as outras bases de dados do módulo de aquisição. Esta numeração se realiza a través do arquivo control_number.cn localizado na pasta bases/suggestions/data.

Trabalhando com multiusuário, é necessário evitar que duas estações de trabalho possam gerar o mesmo número. Para isso não acontecer  proceda da assim:

  1. Verifique se o arquivo control_number.cn tem ou não a propriedade somente leitura
  2. Se não tem a propriedade somente leitura então:
    • Proteja o arquivo para que outra estação não  possa lê-lo.
    • Leia o último número sequencial atribuído e acrescente 1.
    • Faça uma cópia do arquivo control_number.cn (control_number.bak)
    • Acrescente o número sequencial ao campo do número de controle do registro
    • Grave no arquivo control_number.cn o último número sequencial atribuído
    • Desproteja o arquivo control_number.cn para que outra estação possa criar um registro novo
  3. Se o arquivo control_number.cn  tem a propriedade  somente leitura avise esta situação ao operador e forneça um link para tentar de novo a atribuição do número sequencial.

Se após várias tentativas não for atribuído o número sequencial, o Administrador do Sistema deve verificar as condições de permissões do arquivo  control_number.cn  e, se necessário, alterar as permissões para proporcionar o acesso novamente.
