Situação do usuário

Para solicitar a situação da conta do usuário, insira o código do mesmo e clique em Pesquisar.

Também pode utilizar o botão Listar para obter uma lista alfabética dos usuários cadastrados.

Se não souber o código do usuário, pode inserir o número de registro/tombo de algum objeto que esteja em poder do mesmo, utilizando para isso a caixa No. de registro.  Se clicar em Listar serão exibidos os números de registro dos objetos que atualmente estão em poder de algum usuário.
