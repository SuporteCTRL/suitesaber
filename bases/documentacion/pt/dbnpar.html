Archivo dbn.par

O ficheiro dbn.par é um recurso próprio da estrutura CDS/Isis que indica a localização dos recursos de cada base de dados.

Na actual versão do ABCD este ficheiro contem:

  1. O caminho para a directoria onde se encontra a base de dados
  2. O caminho para o formato de início no processo de actualização de cada registo
  3. O caminho para o formato do final do processo de actualização de cada registo

Exemplo:

biblo.*=%path_database%biblo/data/biblo.*
prologoact.pft=%path_database%www/prologoact.pft
epilogoact.pft=%path_database%www/epilogoact.pft

Os formatos de visualização não estão incluidos pois o ABCD gera o caminho a partir dos parâmetros de configuração.

O nome %path_database% é substituido no ficheiro dbn.par pelo caminho actual do separador bases, tal como se especifica na variavel $db_path do script config.php. Este, permite usar a directoria bases sem qualquer impacto nos caminhos dos recursos das bases de dados.

Se pretender colocar os ficheiros da base de dados fora da directoria definida por defeito, altere o ficheiro dbn.par para indicar o caminho absoluto.

Se um formato incluir uma referência a outro formato ou se usa o comando REF para relacionar bases de dados, tanto os formatos que são chamados como as bases de dados relacionadas devem ser indicadas no correspondente ficheiro dbn.par
