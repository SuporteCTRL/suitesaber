Carregar um arquivo Iso

Os archivos ISO que são gerados na exportação são gravados na pasta ./bases/wrk. Para importar um arquivo ISO clique na opção Selecionar que acompanha o ISO a importar.

Se deseja sustituir a base de dados com o resultado dos registros importados, marque a caixa identificada como Apagar todos os registros antes de importar. Se esta caixa não for marcada, os registros importados serão carregados no final da base de dados.

Se deseja atualizar o arquivo invertido enquanto importa os registros, marque a caixa identificada como Gerar arquivo invertido. Se esta caixa não for marcada, deverá gerar o arquivo invertido após o processo com a opção do menu de Manutenção da base de dados.

Nota importante.: Se a base de dados tem muitos registros, recomenda-se usar o utilitário MX para gerar o arquivo invertido

Se o arquivo ISO que se deseja importar não está no servidor, deve-se primeiro subir a arquivo antes de iniciar a importação. Para isso utilize a sessão Subir arquivo ISO que é exibida na tela de importação. Uma vez transferido o arquivo, pode importá-lo.

Nota importante.: o archivo Iso a importar deve ter as mesmas etiquetas e subcampos da base de dados já que não existe uma FST de importação para realizar mudanças na estrutura dos campos.

Se o arquivo Iso a subir é muito grande, deve-se utilizar FTP ou otro meio para subí-lo ao servidor.

Use a opção Apagar que acompanha cada arquivo ISO apresentado, para apagar do servidor os arquivos ISO já utilizados.
