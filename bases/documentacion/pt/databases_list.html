Este procedimento permite alterar a ordenação da lista de bases de dados disponíveis no sistema.

Selecione a base de dados  e clique em ou   até colocá-la no lugar correspondente.

Depois de terminado o processo clique em Atualizar para salvar a nova lista com as alterações.

Deve voltar a entrar no sistema para que este mostre as alterações na lista de base de dados existente na parte superior.

A lista de bases de dados é mantida no arquivo bases.dat localizado na pasta bases.
