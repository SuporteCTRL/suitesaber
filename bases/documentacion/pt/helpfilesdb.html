Ficheiros de ajuda para os campos da base de dados

Para definir as ajudas de cada campo na entrada de dados siga os seguintes passos:

  1. Seleccione no menu Tabela de definição de campos (FDT) o campo que pretende adicionar a ajuda 
  2. Realizada a selecção aparecerá no quadro as caraterísticas do campo, conforme foi definido na tabela de definição de campos(FDT)
  3. Deverá clicar sobre Vista previa para visualizar a ajuda, se esta não estiver presente
  4. Clicar sobre Editar ficheiro de ajuda para criar a ajuda ou modificar a já existente. Será apresentado o editor de HTML para que possa escrever o texto de ajuda
  5. Para terminar a edição da ajuda, clicar sobre Guardar ajuda para actualizar o ficheiro
  6. Os ficheiros de ajuda dos campos são guardados na directoria /bases/dbn/ajudas/tag_xxx.html, onde dbn é o nome da base de dados e xxx representa a etiqueta do campo a que corresponde a ajuda. Este nome é fornecido pelo sistema de forma automática e não deve ser modificado já que serve de referência para o procedimento de entrada da apresentação da ajuda.
  7. Una vez concluidos os ficheiros de ajuda, deve editar a FDT da base de dados para marcar, na coluna correspondente, quais os campos para que foram estabelecidos ficheiros de ajuda.
