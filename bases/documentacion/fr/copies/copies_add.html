Pour ajouter des copies aux objets

Les objets auxquels serait inventé des copies doivent posséder un champ intitulé  "Numéro de Contrôle" qui doit être défini dans la FDT avec le type"autoincrement" et avoir un index dans la FST avec "NC_" un préfixe. Ce champ de la FDT est utilisé comme un champ relationnel entre un objet et leurs copies.

Les copies des objets de toutes les bases de données convergent dans une base de données intitulée "copies"  et maintient un numéro unique d'inventaire pour chaque copie.

Pour l'attribution du numéro séquentiel de contrôle  dans chaque registre, le fichier control_number.cn est utilisé, il est situé dans le dossier  donnéesde la base de données correspondante. Chaque base de données bibliographique possède sa propre séquence de numéro de contrôle. Pour cette raison, dans la base de données "copies"  est stocké le nom de la base de données en relation avec la copie pour garantir la récupération correcte de l'information bibliographique dès que c'est nécessaire.

Quand l'entrée de nouvelles copies est demandée, un formulaire est présenté pour l'entrée de l'information correspondate.

Ecrivez dans le carré, le nombre de copies souhaité à ajouter à l'objet sélectionné. Utilisez le lien "Voir toutes les copies" pour montrer toutes les copies entrées pour cet objet.

Completez les données correspondantes et quand c'est terminé faites cliquez sur "Mettre à jour".

Seront crées dans la base de données "des copies"le nombre d'enregistrements demandés selon la valeur "Nombre de copies". L'information des registres créés prendra la forme des données entrées. Le numéro d'inventaire sera augmenté automatiquement en prenant comme point de départ la valeur stockée  dans les fichiers control_number.cn stockée dans le dossier bases/copies/données .

Utilisez l'option "Rétablir le numéro de contrôle ou d'inventaire" du menu principal du catalogage si pour des raisons il y a besoin de restituer le numéro d'inventaire suivant pour attribuer des copies ou pour les objets d'une base de données particulière.
