Mise en statistiques - Liste des variables

Cette option vous permet de définir des variables dans une base de données qui peut être utilisée pour délivrer des statistiques.

Il exige des trois colonnes de l' information:

Champ Préciser le titre du champ, comme il sera indiqué dans les lignes ou les colonnes de la table de fréquence ou de contingence
Format d' extraction des données
 Le pft à appliquer pour extraire le contenu du champ dans la base de données pour refléter le fait que dans le tableau de fréquence ou de contingence. La valeur produite par le PFT sera utilisée en tant que chef de la ligne ou la colonne et seront accumulées aux fins de l'agrégation
 Préfixe  (Exécution) pour extraire les variables de la liste inversée réduisant ainsi le temps de lire les données
   

Ce fichier est appelé stats.cfg et stocké dans le dossier def de la base de données correspondante.

Les variables définies ici peuvent être utilisées dans deux types de processus:

  • Pour la question des tables de contingence et des tableaux de fréquence, de la sélection des variables qui entrent dans les lignes et les variables qui entrent dans les colonnes
  • Pour la préparation des tableaux pré-definis et dont les résultats sont publiés en sélectionnant le nom de la table correspondante
