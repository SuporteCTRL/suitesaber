Création de nouveaux formats d'exposition

Cette procédure permet de produire un format de sortie ou modifier un précédemment produit. Les formats obtenus peuvent être utilisés pour visualiser l'information dans l´écran ou  envoyer directement l'information à un processeur de mots ou un processeur de feuilles du calcul.

Pour sélectionner un format déjà existant, révisez la liste qui est montrée dans Formats: et cliquez sur le format que vous voulez utiliser. Ensuite cliquez en Éditer que vous présentera une fenêtre avec le format sélectionné. Si vous connaissez les règles pour l´élaboration de formats du Cds/Isis, vous pouvez éditer le format pour faire des changements ou des ajustements; dans un autre façon, ne modifiez pas la sortie parce que s'il produit des erreurs vous n'obtiendrez pas de résultat quand ranger l'émission de la liste.

 Dans la fenêtre où le format s´expose, une fois que les modifications sont faites, cliquez sur envoyer pour envoyer le format à la feuille de génération de sortie.

Pour produire un nouveau format faites click sur la section Créer un format et choisi de la liste de champs ceux que vous voulez inclure dans la sortie. Pour sélectionner les champs faites double click sur le nom ou marquez-les avec l'aide du "souris" et cliquez sur les flèches correspondantes. Vous pouvez utiliser les clefs [Ctrl] et [Shift] pour sélectionner les champs. Les flèches simples transfèrent les champs sélectionnés d'une fenêtre à autres et les flèches doubles transfèrent tous les champs.

Vous pouvez utiliser les boutons monter ou baisser pour changer la place des champs sélectionnés.

Quand la sélection est finie cliquez sur une des options identifiées sous Produir format, et le format produit paraîtra dans la fenêtre. Si vous connaissez les règles pour l´élaboration de formats du Cds/Isis, vous pouvez éditer le format pour faire des changements ou des ajustements; dans un autre façon, ne modifiez pas la sortie parce que s'il produit des erreurs vous n'obtiendrez pas de résultat quand ranger l'émission de la liste.

Une fois complété la sélection/l'édition/la génération d'un format peut se passer à Produire la sortie qui correspond.

Les inscriptions peuvent être sélectionnées de deux manières différentes

  1. À travers une gamme d´enregistre (depuis - jusqu´à)
  2. Appliquer une expression de la recherche sur la base de données. Cliquez sur l'ícono de la recherche et construisez l'expression correspondante

Une fois que la sélection est faite cliquez sur Envoyer à un processeur de mots (pour envoyer les résultats obtenu à un processeur de mots), Envoyer à un processeur de feuilles du calcul, ou sur Vu avant pour voir le résultat du format dans l´écran.

Pour produir la sortie cliquez sur la section Produit Sortie et là placez l'information correspondante selon la méthode qui sera utilisée pour la sélection des enregistre, par n´importe quelle métode, le gamme de Mfn ou la recherche sur la base de données.

Une fois que la sélection est faite cliquez sur Envoyer à un processeur de mots (pour envoyer les résultats obtenu à un processeur de mots), Envoyer à un processeur de feuilles du calcul, ou sur Vu avant pour voir le résultat du format dans l´écran.

Pour sauver le format, cliquez sur la section Sauver le format et donnez un nom (seulement utilisant lettres et nombres et sans espaces en blanc) et fournissez la description avec lequel montrera ce format dans la liste de formats déjà défini.
