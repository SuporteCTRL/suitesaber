Récupération du numéro d'inventaire ou du numéro de contrôle de la base de données


Numéro d'inventaire

Le numéro d'inventaire est une séquence numérique automatiquement gérée par ABCD dans la base d'exemplaires. Ce numéro est défini dans la TDC de la base de données avec le type auto-incrément et est incrémenté à chaque création d'un nouvel enregistrement avec la dernière valeur enregistrée dans la base de données.

La dernière valeur enregistrée est issue du fichier control_number.cn situé dans le dossier data de la base de données concernée.

Cette option modifie la valeur du dernier numéro assigné, afin de rétablir la séquence numérique des enregistrements.

Numéro de contrôle

Le numéro de contrôle est un champ obligatoire dans les bases bibliographiques pour lesquelles les copies seront gérés dans une base exemplaires, dans la mesure où il représente le champ lien entre l'objet bibliographique et ses copies.

Comme dans le cas précédent, ABCD générera automatiquement ce nombre au moyen d'une séquence numérique. De la même façon, le dernier numéro assigné est stocké dans le fichier control_number.cn, situé dans le dossier data de la base de données concernée.

Cette option modifie la valeur du dernier numéro assigné, afin de rétablir le numéro de contrôle des enregistrements.
