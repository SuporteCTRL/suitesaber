Pour générer un format d'extraction (FMT)

Cela vous permet de générer des feuilles de saisie qui peuvent être sélectionnées en utilisant le menu déroulant Feuille d'activités qui se trouve en haut de la fenêtre de ABCD  au moment de la saisie d'information.

Pour créer un nouveau format de saisie :

Sélectionnez les champs qui feront partie de la  fiche de saisie de la liste. Afin de sélectionner un champ vous pouvez :

  • Double clique sur le champ respectif
  • Cliquez sur un champ et puis pour étendre la sélection,  utilisez les clés [Shift]  or  [Ctrl-Shift] et puis sur la flèche
  • Pour copier tous les champs dans la fiche de saisie, cliquez sur

Pour supprimer les champs de la fiche de saisie :

  • Double clique sur le champ respectif
  • Cliquez sur le champ et puis pour étendre la sélection, utilisez les clés [Shift]  or  [Ctrl-Shift] et puis cliquez sur la flèche
  • Pour supprimer tous les champs de la fiche de saisie, cliquez sur

Les champs extraits de la fiche de saisie seront affichés au bas de la fenêtre contenant la liste des champs

Utilisez les boutons  or  pour positionner les champs dans votre fiche de saisie.

Utilisez le lien  pour apercevoir le processus du format de saisie.

Lorsque le fomat est complet, il doit être sauvegardé en incluant le nom du champ où le fichier est stocké, et une brève description sur le format généré. Le nom du fichier du format  peut seulment contenir des lettres, nombres et des caractères _. En outre, le nom du fichier doit toujours commencer par une lettre. Les espaces ne sont pas acceptés  ni les caractères spéciaux. La description doit être raisonnablement courte de façon à ce qu'elle peut être correctement visible dans les menu Feuilles d'activités situées en haut de la fenêtre  principale de ABCD.

Pour modifier un format de saisie existant

Sélectionnez le format que vous voulez éditer dans le menu déroulant puis cliquez sur  Editer. Remplissez les informations dans le format dans lafénêtre des Champs sélectionnés . Suivez les instructions ci-dessous pour ajouter ou supprimer les champs du format.

Pour supprimer un format de saisie existant

Sélectionnez le format que vous voulez dans le menu déroulant et puis cliquez sur  Supprimer pour supprimer.

les formats sont stockés dans le dossier def dans le répertoire de la base de données avec l'extension .fmt. La liste des formats existant est conservée dans le fichier formats.wks. Chaque format de stockage est séparé par le caractère | et contient une ligne pour le nom et la description.

Exemple d'un format généré :

H||Entrées principales|0|0|||||||||||
100
110
111
240
242
245
246
250
260
300
310

Comme on peut le voir, les feuilles de saisie des données contiennent le style du FDT H (en tête) ou L (ligne de séparation) et les étiquettes des champs sélectionnés. Cela veut dire que les modifications effectuées  dans le FDT et la génération ultérieure des feuilles de la saisie des données, seront présentes quand la feuille de saisie des données est sélectionnée.

Le fichier  formats.wks  est lu pour remplir le menu déroulant Feuilles d'activités. Lors de la création ou de la suppression d'un format, le menu déroulant Feuilles d'activités contenu dans le menu de la fenêtre principale est automatiquement mis à jour .
