Cette procédure permet de modifier l'ordre dans lequel est affichée la liste des bases de données disponibles.

Choisissez la base de données et cliquez dessus jusqu'à ce qu'elle soit à la place désirée.

Une fois le processus terminé, cliquez sur Mettre à jour pour sauvegarder les changements apportés à la liste.

Vous devez fermer puis relancer la base de données pour voir les changements dans la liste des bases placée dans le cadre supérieur.

La liste des bases de données est enregistrée dans le fichier bases.dat situé dans le répertoire bases.
