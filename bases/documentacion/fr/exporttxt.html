Exporter la base de données vers un fichier TXT avec les étiquettes Pour exporter une base sous forme d'un fichier TXT avec les étiquettes, on utilise les tables de conversion définies dans l'option Import . (voir tables de conversion) Pour exporter, suivre cette procédure : 1. Sélectionnez la table de conversion à utiliser. Pour ce faire, choisissez l'option Sélection qui apparaît dans chaque table de conversion affichée à l'écran 2. Sélectionnez les enregistrements que vous voulez exporter en utilisant une des options disponibles : * Suite NFM * Recherche * Une lettre du dictionnaire 3. Sélectionnez la façon dont vous souhaitez utiliser pour obtenir le fichier TXT avec les enregistrements exportés en utilisant les deux options disponibles : 4. Traitement de texte 5. Vue précédente Une fois choisis les enregistrements à exporter, inscrivez dans la boîte le nom du fichier ISO à créer et cliquez sur l'icone . Quand le fichier ISO est créé, une fenêtre avec les résultats et un lien vers le nouveau fichier ISO apparaît. Pour télécharger le fichier, faites un clic droit sur le lien pour choisir l'option correspondante. Ensuite, au cours de l'opération d'importation, le fichier ISO obtenu sera affiché comme faisant partie de la liste des fichiers stockés dans le répertoire ./bases/wrk. Note importante : Si la base de données a beaucoup d'enregistrements, il est recommandé d'utiliser l'utilitaire MX pour créer le fichier ISO.
