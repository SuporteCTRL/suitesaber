Assistant au remplissage d'un sous-champ

Utilisez  pour voir le menu d'aide pour le champ

Les occurrences du champ sont affichées dans le menu d'affichage situé à côté du nom du champ.

Utilisez :

 pour ajouter une nouvelle occurrence

 pour supprimer l'occurrence sélectionnée du menu d'affichage

 pour monter l'occurrence sélectionnée dans le menu d'affichage

 pour descendre l'occurrence sélectionnée dans le menu d'affichage

 

Pour créer une nouvelle occurrence, cliquez sur  et utilisez le formulaire qui apparaît dans la fenêtre. Pour éditer une occurrence existant déjà, cliquez dessus dans le menu correspondant.  Pendant l'édition, utilisez :

Le menu des sous-champs pour insérer de nouveaux sous-champs pour l'occurrence active

 Pour monter un sous-champ

 pour descendre un sous-champ

 Pour consulter la liste associée, la liste d'autorité, le thésaurus ou le dictionnaire des termes associés pour le champ correspondant

 

  Quand vous avez fini de modifier l'occurrence, utilisez:

 pour annuler les changements

 pour accepter la nouvelle occurrence ou les modifications sur une occurrence existant déjà. Dans le menu d'affichage vous verrez la nouvelle occurrence ou les changements effectués

 pour passer le champ avec toutes ses occurrences vers le formulaire de saisie
