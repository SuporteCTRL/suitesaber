Pour éditer l'enregistrement en cours. Si l'enregistrement est en train d'être mis à jour par un autre utilisateur, un message vous avertira ce fait, et l'édition demandée ne sera pas activée.

Les enregistrements sont verrouillés pendant la saisie de données pour les protéger contre la mise à jour involontaire et pour éviter l'accès simultané par d'autres utilisateurs. Une fois, la mise à jour terminée, l'enregistremenst est déverrouillé et devient disponible pour les tous les utilisateurs.

Si le cycle de mise à jour n'est pas achevé, c'est-à-dire si l'enregistrement n'a pas accepté de donnée ou si la mise à jour a été arrêtée (boutons Sauvegarder ou Annuler dans la barre d'outils), l'enregistrement reste verrouillé pendant un certain temps (4 heures) pour les autres utilisateurs.

L'enregistrement peut être déverrouillé avant ce délai, de deux façons:

  1. L'utilisateur qui a verrouillé l'enregistrement peut le libérer (ou déverrouiller) en faisant la mise à jour, l'édition ou l'annulation de cette édition.

  2. En utilisant l'option "Déverouiller les enregistrements" disponible dans le menu Utilitaires dans la barre d'outils. Seuls l'Administrateur du Système ou Administrateur de Base de Données ont accès à cette option.

Menu des formats d'affichage  

Les formats d'affichage (PFT) préparés par l'Adminsitrateur de Base de Données sont affichés dans ce menu. Les autres formats sont affichés à la fin de la liste:

  • Sans format: indique que les champs seront affichés dans un tableau, avec les étiquettes définies dans la TDC

  • Tout: montre les enregistrements sans aucun format d'impression. C'est l'équivalent du format ALL des structures CDS/Isis

Menu des Bordereaux de saisie

Les bordereaux de saisie définis par l'Adminsitrateur de Base de Données sont affichés. Si un bordereau est sélectionné, il restera actif pour la crétaion et la mise à jour des enregistrements qui suivent.
