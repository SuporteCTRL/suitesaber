Création du format d'exposition

La liste de champs existant est exposé selon le tableau de définition de champs.

Choisissez dans la fenêtre de gauche les champs à utiliser dans la exposition du format et passe-les à la fenêtre droite en utilisant les flèches de direction. Utilisez les links monter y baisser pour modifier l'ordre des champs dans le format à produire.

Une fois que la sélection est faite indiquez si vous voulez produire la sortie dans la forme du tableau (separando los datos en filas y columnas) ou dans la forme de paragraphe (séparer les données avec un saut de la ligne). Le format produit sera montré dans la fenêtre.

Cliquez sur le link Créer la base de données pour créer la base de données.
