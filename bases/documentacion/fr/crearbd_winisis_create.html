Créer une nouvelle base de données à partir d'une base WinIsis existante

Il est possible de créer une base de données en utilisant les TDC(FDT), TSC(FST) et PFT déjà créés pour une base de données locale avec WinIsis.

ABCD permet d'importer la TDC(FDT) depuis l'ordinateur local et de la convertir vers la structure TDC(FDT) stockée dans le nouveau système.

Il est aussi possible d'importer la TSC(FST) et un des formats d'affichage (PFT).

Notez bien que le format d'affichage sélectionné doit être écrit en utilisant des littéraux préfixes et suffixes afin de reproduire correctement une page HTML. Les formats d'affichage contenant des éléments utilisés pour la visualisation sous WinIsis, et non dans un navigateur, produiront une erreur de syntaxe lors de leur visualisation sous ABCD.

Une fois ces trois étapes terminées, ABCD va créer la base de données et l'ajoutera dans la liste des bases existantes.

Afin de permettre à la base de données d'être utilisée par d'autres que l'administrateur système, il est nécessaire d'activer l'option Gestion des utilisateurs afin de créer ou sélectionner les utilisateurs autorisés et définir les droits appropriés.
