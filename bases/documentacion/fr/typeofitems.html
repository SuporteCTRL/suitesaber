Types des articles

Cette table décrit les règles appliquées aux différents types d'articles existants dans la base de données. The type d'objet dont il est question dans la table devrait être généré par le format d'extraction appliqué à la base de données contenant les objets en prêt ; cette sortie, parmi d'autres données, devrait contenir le type d'objet qui sera utilisé dans la table pour définir les règles de gestion correspondantes.

Si le format d'extraction ne génère aucune sortie alors le gestionnaire du prêt décide du type de prêt à appliquer en sélectionnant une des options de la table Types de prêts.

Quand l'enregistrement d'un prêt est fait, le type d'objet ou le type de prêt sont stockés afin d'assurer l'application correcte des règles de retours, d'amendes ou de suspension.

Description de la table

Champ Usage dans l'application des règles de gestion
Type d'objet Valeur issue du format d'extraction format apliqué à la base de données bibliographique.
Description Nom du type d'article
Type d'utilisateur Spécifie le type d'utilisateur auquel s'applique les régles pour un article de ce type. Si aucun utilisateur n'est sélectionné, les règles s'appliques à tous les utilisateurs
Période de prêt Nombre de jours/heures pour calculer la période de prêt
Unité Unité de mesure des périodes de prêt (Jours/Heures)
No. de réservations Nombre de réservations autorisées
Amende par jour de retard Affiche l'amende comptabilisée pour chaque jour/heure de retour pour le retour d'un prêt
Jours de suspension par jour de retard Précise combien de jours l'utilisateur sera suspendu pour chaque jour/heure de retard au retour
Réservé. Amende par jour de retard Si l'objet est réservé, spécifie le montant de l'amende par jour de retard à appliquer au retour
Réservé. Suspension par jour de retard Si l'objet est réservé, spécifie le nombre de jours de suspension par jour de retard à appliquer au retour
