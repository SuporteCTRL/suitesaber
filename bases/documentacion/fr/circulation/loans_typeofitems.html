Types de matériel

Cette table décrit les politiques à appliquer aux différents types de matériel disponibles dans la base de données. Le type de matériel auquel l'on se refère dans la table doit être générée par le format d'extraction appliqué à la base de données des articles qu'on peut emprunter; cette production doit contenir entre autres données, le type de matériel à localiser dans la table afin d'attribuer la politique respective.

Si le format d'extraction ne donne aucun resultat, alors, l'opérateur des prêts déterminera le type de prêt à appliquer en sélectionnant une option à partir de la table Types de prêt.

Quand l'enregistrement du prêt est sauvegardé, c'est le type de matériel ou le type de prêt qui lui a été attribué en vue d'appliquer correctement les politiques de retour/d'amende/de suspension.
