Paramètres requis pour configurer une base de données avec des articles empruntables

1. Préfixe pour localiser le numéro d'inventaire

Ce préfixe est ajouté au numéro d'inventaire fourni par l'opérateur des prêts en vue de localiser le titre approprié. Il est stocké dans le dossier de la base de données sélectionnée def sous le nom loans_conf.tab
2. Préfixe pour localiser le numéro de la classe Ce préfixe est ajouté au numéro de la classe fourni par l'opérateur des prêts en vue de localiser le titre approprié.  Il est stocké dans le dossier de la base de données sélectionnée def sous le nom loans_conf.tab file.
3. Pft pour calculer le nombre total de copies de l'article

Ce format est appliqué pour calculer le nombre total de copies de l'article à emprunter après qu'il a été localisé par les moyens de l'une des méthodes précedentess. Il est stocké dans le dossier de la base de données def sous le nom loans_totalitems.pft
4. PFT pour extraire le numéro d'inventaire Ce format est appliqué sur l'article à emprunter pour extraire leurs numéros d'inventaire et localiser la copié désirée. Il est stocké dans le dossier de la base de données sélectionnée def sous le nom loans_inventorynumber.pft
5. PFT pour extraire le numéro de la classe
6. PFT pour afficher le registre de l'article à emprunter Ce format est appliqué au registre situé dans la base de données avec les articles empruntables en vue de les visualiser.Il est stocké dans le dossier de la base de données sélectionnée def sous le nom loans_display.pft
7. PFT pour avoir le type de registre à partir de la base de données des prêts Ce format est appliqué au registre situé dans la base de données avec les articles empruntables en vue d'avoir le type de registre des copies exigées. Ce type de registre est lié au type d'usager pour déterminer la politique de prêt de la copie exigée Il est stocké dans le dossier de la base de données selectionnée def sous le nom loans_typeofobject.pft
8. PFT pour stocker le registre dans la base de données des prêts Ce format est appliqué au registre situé dans la base de données des prêts et il est utilisé pour stocker l'information catalographique de la copie empruntée. Il est stocké dans le dossier de la base de données sélectionnée def sous le nom  loans_store.pft
9. PFT pour afficher l'article de la base de données des prêts Ce format ets utilisé pour afficher l'information sur les prêts en attente dans la base de données des prêts. Il est stocké dans le dossier de la base de données sélectionnée def sous le nom loans_show.pft

Vous pouvez utiliser le lien Essayer et entrer un MFN pour appliquer les formats appropriés à la base de données avec les articles empruntables.

Pour stocker les formats, cliquez sur Sauvegarder
