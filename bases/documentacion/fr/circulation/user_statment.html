Pour demander un relevé de compte, écrivez le code d'utilisateur et  cliquez sur Rechercher. Vous pouvez utiliser le bouton Feuilleter pour une liste alphabétique des utilisateurs enregistrés.
Si vous ne connaissez pas le code de l'utilisateur, vous pouvez fournir le numéro d'inventaire d'un objet qui est tenue par lui même, en utilisant la case Inventaire .  Si vous cliquez sur Feuilleter il permet de présenter les numéros d'inventaire des objets qui sont actuellement détenues par un utilisateur.
