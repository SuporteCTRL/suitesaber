Types de prêt

Cette table est appliquée lorsque le format d'extraction pour les articles disponibles pour le prêt ne fournit aucun article disponible dans la table des Types de matériaux.

Il sera affiché à l'opérateur qui déterminera la politique à appliquer.

Field Comment appliquer la politique
Type de prêt Code du type de prêt
Description Type de la description du prêt
Période du prêt Nombre de jours/heures pour calculer la date de retour.
Unité Unité de mesure de la période de prêt (jours/heures)
Nombre de renouvellements Nombre de renouvellements autorisés
Amende par jour après la date de retour. Le montant de l'amende encourue par jour/heure après la date de retrour des artciles.
Nombre de jours de suspension par jour après la date de retour Il indique le nombre de jours pour lequel l'usager sera suspendu par jour/heure après la date de retour des articles.
