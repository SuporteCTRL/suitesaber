Copier des enregistrements de serveurs distants en utilisant le protocole Z39.50

Cette option permet dans ABCD de catalogaguer directement à partir d'autres sources disponibles à travers le protocole Z39.50.

Sous le menu  Connecter à :   est affichée une liste de serveurs qui sont disponibles par la localisation des serveurs. Cette liste est maintenue à travers l'option Configurer le client Z39.50 disponible (à l'administrateur) dans le menu principal de ABCD.

Le menu Table de conversion contient les systèmes qui ont été définis pour convertir les enregistrements du format MARC  vers la définition des champs dans la base de données locale qui recevra les données. Cette liste est mainttenue à travers l'option Configurer le client Z39.50 accessible (à l'admiistrateur) dans le menu principal de ABCD. Si aucun système de conversion n'est sélectionné, la FDT de la base de données est utilisée pour déterminer les champs à copier de la base de données à distance.

Utilisez les options de recherche pour établir les critères à appliquer pour personnaliser les informations que vous voulez cataloguer.

Dans la boîteMontrer indiquez combien seront pris en compte pour configurer les pages avec les résultats obtenus.

Essayez xx fois la connexion spécifies le nombre de fois que le système ressaye de se connecter quand le serveur ne peut être trouvé.

Dans les conditions normales, les enregistrements transférés créent de nouveaux enregistrements dans la base de données locale qui reçoit les données.  Pour transférer les données à un enregistrement déjà existant, suivez la procédure ci-dessous :

  1. Localisez dans la base de données locale l'enregistrement existant qui doit recevoir les données du catalogue à travers Z39.50.
  2. Après avoir localisé l'enregistrement, cliquez sur le bouton éditer sur la barre d'outils. 
  3. Quand le formulaire pour éditer l'enregistrement est montré, cliquez sur le bouton qui appelle le client Z39.50. Si l'enregistrement est en cours de modification, vous devez sauvegarder l'enregistrement avant d'appeller le client Z39.50 puisque l'autre formulaire perdra les modifications effectuées.
  4. Sélectionnez la base de données suivante à partir de laquelle vous allez cataloguer, le format de conversion (si nécessaire) et formulez la recherche.
  5. In the results presented, locate the record you wish to capture.
  6. Cliquez sur l'icône de Capture. Les données extraites du serveur Z39.50 sont copiées dans l'enregistrement sélectionné :
    • Convertissez les données en appliquant le format de conversion (s'il est sélectionné)
    • Transférez les données aux champs de la base de données locale, seulement si les mêmes sont vides, c'est-à-dire que cette opération de copie n'a pas remplacé les données déjà existantes dans l'enregistrement.
  7. Sauvegardez l'enregistrement dans la base de données locale.
