Table de conversion pour l'exportation de base de données vers un fichier TXT avec des étiquettes

La table de conversion est utilisée pour l'exportation et l'importation d'enregistrements, répondant dans les deux cas aux caractéristiques suivantes:

Utiliser la boîte Délimitée par Tabulation pour importer une feuille EXCEL sauvegardée en TXT. Dans cette case, la colonne Étiquette dans le fichier TXT doit contenir le numéro de la colonne (commençant par 1) dans laquelle le champ correspondant se trouve dans la feuille Excel exportée.

La table de conversion contient les colonnes suivantes:


Étiquette dans le fichier TXT

Étiquette qui identifie le champ dans le fichier TXT en entrée ou en sortie. Les étiquettes doivent commencer par $$ et se terminer par :

Étiquette

Étiquette du champ dans le fichier (la base de données)

Sous-champs

Délimiteurs de sous-champ. Laisser cette case vide si le premier champ n'a pas de délimiteur.

Remplacer par

Si le champ a des sous-champs, indiquer les caractères qui seront utilisés pour remplacer les marques de sous-champ dans le texte. Le premier caractère de cette liste remplacera le premier sous-champ dans la liste des sous-champs. Laisser cette case vide si le premier sous-champ ne sera pas à remplacer par un délimiteur.

Séparateur d'occurrence

Indiquer ici le caractère qui séparera les occurrences d'un champ répétitif.

Format d'extraction

Ce format agit sur l'enregistrement pour donner une sortie du champ associé. L'application du format d'extraction est exécutée seulement pendant l'importation. Si vous utilisez à la fois des délimiteurs de sous-champs et un format d'extraction, ce dernier sera le format appliqué pendant l'exportation (non encore installé).
