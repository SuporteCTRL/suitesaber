Le module des achats repose sur trois bases de données:

  1. Recommandations
    Il contient les demandes des utilisateurs pour les besoins de l'acquisition de nouveaux objets ou de nouvelles copies d'objets existants. Il contient 4 groupes d´information:
    • Les données de l'objet
    • Les données de la personne ou du service objet de la demande
    • Les données de l'approbation ou le rejet de la demande
    • Les données du devis pour l'achat de l'objet
    • Les données de l'ordre d'achat
    • Revendications
  2. Fournisseurs
    Grâce aux informations fournies par les fournisseurs qui participent au processus de cotation et l'achat d'articles
  3. Copies
    Contient de chacun des exemplaires qui ont été acquises dans le cadre du processus d'approvisionnement.

Le processus à suivre est le suivant:

  1. Il apparaît un nouveau bon de commande. Cette demande peut traîter l'achat d'un nouvel objet ou l'acquisition de nouvelles copies d'un objet existant.
    Chaque objet dans la base de données est identifié par un numéro unique qui pourra ensuite établir la relation entre l'objet et de ses copies.
    Chaque exemplaire aura également un numéro d'inventaire, qui est utilisé plus tard sous dans le module de circulation.
    Si l'objet existe déjà dans la base de données, la demande doit indiquer le numéro attribué à cet objet. Si un nouvel objet, puis le numéro sera attribué lors de catalogage objet, après l'achat.
  2. Chaque proposition doit être approuvée ou rejetée. S'il est approuvé, il sera nécessaire  d'indiquer le nombre d'exemplaires à acheter
  3. Les demandes approuvées sont soumises à un processus de négociation, en notant dans le registre des fournisseurs et de prix.
  4. Avec les informations des cotes on décide l'achat; celle-ci est insérée dans la base de données du fournisseur d'information.
  5. On émis des bons de commandes à des fournisseurs sélectionnés
  6. S'il y a des retards dans l'arrivée du matériel, on enregistre sur la base de données les différentes demandes.
  7. Quand il arrive le matériel acheté , il existe deux options:
    • Si l'objet est nouveau, doit être attribué le numéro d'identification, cataloger l'objet et créer un dossier pour chaque copie de la base de   données exemplaires, donnant le numéro d'inventaire
    • Si l'objet existe déjà dans la base de données, il suffit d'ajouter un nouvel enregistrement dans la base de données sur le nombre de copies de l'inventaire
