Créer une nouvelle recommandation

Avant cela insérez dans la base de données une nouvelle recommandation, verifiez si elle est une nouvelle acquisition complémentaire d'un objet. Dans ce cas, il est nécessaire de donner la base de données où l'information de catalogage de l'objet et le numéro d' identification de l'objet, telle qu'elle a été attribuée dans la base de données correspondante.

Si c'est un nouvel objet, vous pouvez aussi donner le nom de la base de données où l'objet sera classé.

Les données obligatoires sont :

Etiquette   
2 Etat de la
3 Type d'acquisition
5 Base de données ou l'objet est classé
6 Numéro d'identification de l'objet (Obligatoire si  le type d'identification sélectionnée est Copies complémentaires)
16/17 Indiquez l'auteur personne ou l'auteur organisme
18 Titre
200 Nom de la personne ou du département qui a suggéré le bon de commande
210 Date de suggestion
211 Date ISO de suggestion
220 Nombre de copiés suggéré

L'information entrée est sauvegardée dans suggestions base de données  et aux nouveaux enregistrements sont attribués dans un formulaire automatique un numéro séquentiel qui est utilisé comme un champ relationnel dans le processus de suivi de l'état de la recommandation.

Pour l'attribution de numéro séquentiel, retez dans le dossier données des suggestions de base de données un control_number.cn dénommé fichier qui a gravé le dernier numéro attribué.
