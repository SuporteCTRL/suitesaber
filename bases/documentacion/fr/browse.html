Autorisation d'accès en ABCD 

Les utilisateurs ou opérateurs ABCD sont des personnes qui ont accès aux différentes fonctions et bases de données comme définis dans le système. Il y a 8 types d'opérateurs:

Administrateurs de Système

Ils ont accès à tous les modules, fonctions et bases de sonnées définis dans ABDC. Ce sont les Administrateurs de Système qui créent les nouveaux utilisateurs et en déterminent les autorisations d'accès.

Administrateurs de Base de Données

Ils peuvent :

  • Créer, modifier et supprimer les structures de base de données
  • Effectue toutes les tâches relatives à la gestion des bases de données sous leur contrôle (modifier la structure, les TSC, configuration des statistiques, débloquer des bases de données et des enregistrements, bloquer les base pour arrêter la saisie de sonnées, export et import d'enregistrements)
  • Effectuer des changements globaux sur les bases de données sous leur contrôle
  • Attribuer à des Operateurs les bases de données sous leur contrôle

Operateurs de Base de Données

Ils peuvent:

  • Créer et modifier des enregistrements dans les bases de données sous leur contrôle
  • Générer des rapports et d'autres services dans les bases de données sous leur contrôle

Gestionnaires des Prêts

Ils peuvent:

  • Définir et Modifier la politique de prêt
  • Effectuer toutes les fonctions se rapportant au système de prêt
  • Gérer les bases de données du système de prêt: transactions, réservations, suspensions et amendes
  • Supprimer les réservations, les amendes et les suspensions

Operateurs des Prêts

Ils peuvent:

  • Effectuer toutes les transactions concernant le service de prêt
  • Obtenir des rapports et des produits de prêt

Administrateur des Thésaurus

Dans la fenêtre, vous trouverez les opérateurs actuellement définis dans ABCD. Pour supprimer un opérateur, ajouter l'icone Éliminé dans la ligne correspondante. En cliquant sur l'icone d'édition, vous pouvez réutiliser l'enregistrement.

Créer un Nouvel Opérateur

Cliquer sur Nouvel Utilisateur pour créer un nouvel opérateur ABCD. Un formulaire avec les champs nécessaires est affichée. Remplir les informations nécessaires et cliquer:

  • Sauvegarder l'enregistrement, ou
  • Annuler

selon l'action voulue. [début]

Modifier un Opérateur

Cliquer l'icone d'édition associé à l'opérateur que vous voulez modifier. Un formulaire avec les champs nécessaires est affiché. Remplir les informations nécessaires et cliquer Sauvegarder ou Annuler selon l'action voulue. [début]

Supprimer un Opérateur

Cliquer l'icone de suppression associé avec l'opérateur que vous voulez supprimer. [début]
