Exporter la base de données vers un ficher ISO ou TXT

les fichiers ISO ou les fichiers TXT qui sont créés par les fonctions d'exportation sont stockés dans le dossier ./bases/wrk. Pour exporter une base de données et obtenir un fichier d'exportation, sélectionnez les enregistrements que vous voulez exporter par les options disponibles :

  • Suite NFM

  • Recherche

  • Une lettre du dictionnaire

Une fois sélectionné les enregistrements à exporter, insérez dans la boîte le nom du fichier ISO ou TXT que vous voulez créer et cliquez sur .

Quand le fichier est créé, une page avec les résultats et un lien vers le fichier apparaît. Si vous voulez télécharger le fichier, faites un clic droit sur le lien pour voir l'option qui vous permettra de le faire.

Ensuite, lors de l'importation, le fichier que vous avez obtenu dans le processus d'exportation sera affiché comme faisant partie de la liste des fichiers stockés dans le répertoire ./bases/wrk.

Note importante : Si la base de données a beaucoup d'enregistrements, il est recommandé d'utiliser l'utilitaire CISIS pour créer les fichiers ISO ou TXT.
