Types d'enregistrements (Format MARC)

Cette option permet de définir les types d'enregistrements qui seront gérés par la base de données lorsque le système utilise le format MARC.

The type d'enregistrement se trouve dans le champ 3006 de l'en-tête et permet d'identifier le format du champ fixe (008) à utiliser dans la description bibliographique du document.

Définir le type d'enregistrement requiert deux paramètres :

  1. Identifier le champ (étiquette) de la base de données dans laquelle on trouvera (ou sera entré dans le cas d'un nouvel enregistrement) l'enregistrement correspondant au document. La version actuelle d'ABCD permet la définition de deux champs dont les valeurs détermineront le format du champ fixe (008) à afficher dans le formulaire de saisie des données.
  2. Décrire le type d'enregistrement avec les paramètres suivants 
    • Nom de la table de définition des champs (TDC) qui contient les éléments utilisés pour générer le formulaire de saisie des données.
    • Valeur des champs identifiés en (1) qui sont considérés comme appartenant à ce type d'enregistrement
    • Description du type d'enregistrement qui sera affiché à l'utilisateur lors de la saisie de données avant de sélectionner le type correspondant

Afin de maintenir la cohérence dans les définitions d'ABCD, les champs fixes sont définis par une TDC qui contient la description des éléments de données associés à un type d'enregistrement et avec toutes les caractéristiques définies lors du processus de création de la TDC de la base de données. Pour éditer la FDT associée aux champs fixes pour type d'enregistrement, cliquer sur le lien Editer. Cela ouvrira une nouvelle fenêtre où les caractéristiques des éléments de données, y compris les champs fixes, seront affichés.
