Modifier les définitions d'une base de données

Une fois qu'une base de données a été créée, il est possible d'entrer dans ce menu pour modifier n'importe lequel des composants d'une base de données ABCD.

Les composants existants sont :

  • Table de définition des champs. Définit la structure des enregistrements de la base de données
  • Table de sélection des champs. Définit les règles à appliquer pour contruire les index permettant d'accéder aux registres de la base de données
  • Formats de saisie. Formats de saisie supplémentaires que l'on peut utiliser pour la saisie d'information
  • Formats de sortie. Formats d'affichages que l'on peut utiliser pour visualiser l'information.
  • Types d'enregistrements. Définit les formats de saisie pour chacun des types d'enregistements définis dans la base de données (si ils existent)
  • Formats de validation d'enregistrements. Permet de faire une vérification de la cohérence d'un enregistrement avant de le sauvegarder
  • Formulaire de recherche avancée. Définit les champs qui pourront être utilisés lors de la recherche avancée
  • Liste des bases de données disponibles. Liste des bases de données présentes dans les différents modules d'ABCD
  • Fichier dbn.par. Chemin d'accès aux ressources d'une base de données
  • Fichiers d'aide pour les champs de la bases de données. base de datos. Aide en ligne disponible lors de la saisie des informations
  • Configuration sous IAH. Configuration de la base de données pour le site de consultation publique.
  • Liste de variables pour l'obtention de statistiques. Liste des champs que peuvent être utilisés pour obtenir des statitistiques, indiquer le format d'extraction pour chacun
  • Tableaux statistiques. Combinaison de champs en rangées et colonnes pour générer des tableaux pré-définis
