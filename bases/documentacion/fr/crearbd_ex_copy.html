Créer une nouvelle base de données à partir des ressources d'une base existante

Cette option crée les fichiers de définition d'une nouvelle base de données à partir des ressources d'une base de données existante.

Le processus vérifie tous les répertoires de la base modèle et les copie dans un répertoire créé du nom de la nouvelle base de données.

Le fichier .par de la nouvelle base de données est créé et la base de données est ajoutée au fichier bases.dat

Une fois le processus de création terminé, vous devez assigner les permissions pour les utilisateurs qui administreront la base de données. Sans cela, elle sera uniquement accessible à l'administrateur système.
