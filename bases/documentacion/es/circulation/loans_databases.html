Bases de datos objeto de préstamo.

Mediante este procedimiento se pueden seleccionar las bases de datos definidas bajo ABCD que tienen objetos disponibles para préstamo a los usuarios del servicio.

Seleccione de la lista presentada una base de datos y luego haga clic sobre Ir para definir los parámetros y formatos requeridos a efectos de la conexión entre la base de datos y el sistema de préstamos.

En líneas generales se requiere definir:

  • La clave de acceso a los títulos
  • Como determinar el total de ejemplares del título
  • Como extraer la información de cada ejemplar
  • Como presentar la información catalográfica del título
  • La información del título  a transferir a la base de datos de préstamos, para identificación de los materiales prestados y para la obtención de las estadísticas
  • Como presentar la información del título prestado, una vez que haya sido almacenado en la base de datos de préstamos

Una vez seleccionada la base de datos se presentará un formulario de ingreso donde se solicitan los parámetros y formatos requeridos por el proceso
