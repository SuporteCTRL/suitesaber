Tipos de material

Esta tabla describe las políticas a aplicar para los diferentes tipos de material existentes en la base de datos. El tipo de material  referenciado en la tabla debe ser generado por el formato de extracción que se aplica sobre la base de datos con los materiales objeto de préstamo; esta salida, entre otros datos, deberá contener el tipo de material a ubicar en la tabla para asignarle la pólítica respectiva.

Si el formato de extracción no genera ninguna salida, entonces, el operador de préstamo decidirá el tipo de préstamo a aplicar seleccionando una de las opciones de la tabla Tipos de Préstamo.

Cuando se graba el registro de préstamo, se almacena el tipo de material o de préstamo que le ha sido asignado a efectos de aplicar correctamente las políticas de devolución/multa/suspensión

Se almacena en la carpeta bases/circulation/def/lang bajo el nombre items.tab
