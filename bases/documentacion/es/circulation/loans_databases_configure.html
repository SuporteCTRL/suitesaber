Parámetros requeridos para configurar la base de datos con los objetos de préstaqmo

1. Prefijo para localizar el número de inventario

Este prefijo se agrega al número de inventario suministrado por el operador de préstamos, a fin de localizar el titulo correspondiente.  Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_conf.tab
2. Prefijo para localizar el número de clasificación Este prefijo se agrega al número de clasificación suministrado por el operador de préstamos, a fin de localizar el titulo correspondiente.  Se almacena en la carpeta def de la base de datos seleccionada en el mismo archivo loans_conf.tab 
3. Pft para calcular el total de ejemplares del título Una vez localizado el ítem objeto de préstamo, se aplica este formato sobre el registro localizado por uno de los dos métodos anteriores y con el mismo obtiene el total de ejemplares del título.  Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_totalitems.pft
4. PFT para extraer el número de inventario Este formato se aplica sobre el ítem objeto de préstamo a fin de extraer los números de inventario del título e identificar el ejemplar solicitado. Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_inventorynumber.pft
5. PFT para extraer el número de clasificación
6. PFT para desplegar el registro objeto del préstamo Este formato se aplica sobre el registro localizado en la base de datos con los objetos de préstamo, a efectos de visualizar el registro.Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_display.pft
7.  PFT para obtener el tipo de registro de la base de datos con los objetos de préstamo Este formato se aplica sobre el registro localizado en la base de datos con los objetos de prestamo a fin de obtener el tipo de registro del ejemplar solicitado. Este tipo de registro se combina con el tipo de usuario para determinar la política de préstamo del ejemplar solicitado.Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_typeofobject.pft
8. PFT para guardar el registro en la base de datos de préstamos Este formato se aplica sobre el registro localizado en la  base de datos de préstamos y se utiliza para guardar la información catalográfica del ejemplar prestado al usuario. Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_store.pft
9. PFT a utilizar para desplegar el objeto desde la base de datos de préstamos Este formato se utiliza para poder desplegar la información de los prestamos pendientes desde la base de datos de préstamos. Se almacena en la carpeta def de la base de datos seleccionada, bajo el nombre loans_show.pft

Puede usar el enlace Probar y suministrar un Mfn para aplicar los formatos suministrados sobre la base de datos con los objetos de préstamo.

Para almacenar los formatos, haga clic sobre Guardar.
