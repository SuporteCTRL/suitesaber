Tabla de conversión para exportación de la base de datos hacia un archivo TXT con rótulos

La tabla de conversión se utiliza indistintamente para exportar o para importar registros, aplicandose en cada caso las siguientes modalidades:

Use la casilla  Delimitado con tabuladores, para importar una hoja excel almacenada con TXT. En ese caso la columna Etiqueta en el archivo TXT deberá contener el número de la columna (comenzando por 1) donde se encuentra el campo correspondiente en la hoja Excel exportada

Consta de las siguientes columnas:
Etiqueta en el archivo TXT Etiqueta con que se identifica el campo en el archivo TXT de entrada o de salida. Las etiquetas deben comenzar con $$ y terminar con :
Tag Etiqueta del campo en la base de datos
Subcampos Delimitadores de subcampos. Si el primer subcampo no tiene delimitador, dejar un espacio en blanco
Reemplazar por Si un campo tiene subcampos, indicar aquí los caracteres de edición con los cuales se van a reemplazar los subcampos del campo. El primer caracter de esta lista reemplazaría al primer subcampo de la lista de subcampos. Si el primer subcampo no va a ser sustituído por ningún delimitador, dejar un espacio en blanco
Separador de ocurrencias Indique aquí el caracter con el cual se separan las ocurrencias de un campo repetible
Formato de extracción Este formato se aplica sobre el registro para generar la salida del campo asociado. La aplicación del formato de extracción solo se ejecuta en el proceso de exportación.Si se suministran los delimitadores de subcampo y el formato de extracción, este último es el que aplica en el proceso (no implementado)
