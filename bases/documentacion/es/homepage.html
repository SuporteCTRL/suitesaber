Crear base de datos

  • Subiendo las definiciones desde una base de datos definida en WinIsis
  • A partir de las definiciones de otra base de datos ya definida en ABCD
  • Nueva base de datos

Restablecer el último número de inventario o el número de control
 Si la base de datos tiene definido un campo del tipo de incremento automático a efectos de almacenar un número de control a cada registro, esta opción permite alterar la secuencia modificando el valor del último número asignado en el archivo control_number.cn
Si la base de datos tiene control de copias, permite modificar el valor del último número asignado en el inventario de las copias
