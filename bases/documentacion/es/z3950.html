Copiar registros desde servidores remotos utilizando el protocolo Z39.50.

Esta opción de ABCD le permite catalogar en forma directa registro provenientes de otras fuentes disponibles a través del protocolo Z39.50.

Bajo el menú  Conectar a:  se muestra la lista de servidores que están disponibles para la localización de servidores. Esta lista se mantiene a través de la opción Configurar cliente Z39.50 disponible (para el administrador) en el menú principal de ABCD.

El menú Tabla de conversión contiene los esquemas que han sido definidos para convertir los registros desde el formato MARC  hacia la definición de campos de la base local que va a recibir la información. Esta lista se mantiene a través de la opción Configurar cliente Z39.50 disponible (para el administrador) en el menú principal de ABCD. Si no se selecciona ningún esquema de conversión se utiliza la FDT de la base de datos local para determinar los campos que van a copiarse desde la base de datos remota.

Utilice las opciones de búsqueda para establecer los criterios a aplicar para localizar la información que desea catalogar.

En el recuadro Mostrar indique cuántos se van a considerar para configurar las páginas con los resultados obtenidos

Intentar xx veces la conexión se establece cuántas veces reintentará el sistema establecer una conexión en caso de que el servidor no pueda ser localizado.
