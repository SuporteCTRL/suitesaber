Generar una nueva base de datos a partir de la copia de los recursos definidos para otra base de datos

Esta opción genera los recursos necesarios para definir una nueva base de datos, a partir de los recursos ya definidos para otra base de datos ya existente.

El procedimiento recorre las carpetas de la base de datos seleccionada como modelo y los copia en una carpeta creada con el nombre de la nueva base de datos.

Se crea el archivo .par para la nueva base de datos y se incluye la misma en el archivo bases.dat

Una vez finalizado el proceso de creación, debe asignar los operadores que pueden gestionar la nueva base de datos, ya que otra forma sólo estará disponible para el Administrador del sistema
