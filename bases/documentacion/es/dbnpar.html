Archivo dbn.par

El archivo dbn.par es un recurso propio de las estructuras CDS/Isis que indica la localización de los recursos de una base de datos.

En la versión actual de ABCD este archivo contiene:

  1. La dirección hacia la carpeta donde se encuentra la base de datos
  2. La dirección hacia el formato que construye el prólogo en el proceso de actualización de un registro
  3. La direccón hacia el formato que construye el epílogo del proceso de actualización de un registro

Ejm:

biblo.*=%path_database%biblo/data/biblo.*
prologoact.pft=%path_database%www/prologoact.pft
epilogoact.pft=%path_database%www/epilogoact.pft

Los formatos de salida no se incluyen en este archivo ya que ABCD construye la ruta directamente en base a los parámetros de configuración.

El nombre %path_database% es sustituído en el archivo .par por la dirección actual de la carpeta bases, tal como se especifica en la variable $db_path del script config.php. Esto permite movilizar la carpeta bases sin causar impacto en las direcciones de los recursos de las bases de datos.

Si desea colocar los archivos de una base de datos fuera de la carpeta base, indique en el archivo dbn.par su dirección absoluta.

Si un formato de despliegue hace referencia a otro formato o se usa la sentencia REF para relacionar bases de datos, tanto los formatos llamados, como las bases de datos relacionadas deben ser agregadas al archivo dbn.par correspondiente
