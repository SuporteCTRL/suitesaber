Permisos en ABCD 

Los operadores o usuarios de ABCD son aquellas personas que poseen acceso a las diferentes funciones y bases de datos definidas bajo el sistema. Existen 8 tipos de Operadores:

Administrador del Sistema

Tiene acceso a todos los módulos, funciones y bases de datos definidos bajo ABCD. También es el encargado de definir nuevos usuarios y asignarles la permisología

Administradores de Bases de Datos

Pueden:

  • Crear, modificar y eliminar estructuras de bases de datos
  • Realizar todas las funciones de administración sobre las bases de datos que están bajo su control (modificar estructuras, fst's, parámetros de configuración de estadísticas, desbloquear bases de datos y registros, bloquear bases de datos para impedir el ingreso de información, exportar e importar registros
  • Realizar procesos de cambios globales sobre las bases de datos que están bajo su control
  • Asignar Operadores para las bases de datos que están bajo su control

Operadores de Bases de Datos

Pueden:

  • Realizar funciones de creación y edición de registros en las bases de datos que están bajo su control
  • Emitir reportes o generar otros servicios en las bases de datos que están bajo su control

Administrador de Préstamo

Puede:

  • Crear/Modificar las políticas de préstamo de la institución
  • Ejecutar todas las funciones relacionadas con el sistema de préstamos
  • Administrar las bases de datos de transacciones, reservas y suspensiones y multas
  • Eliminar reservas, multas y suspensiones

Operadores de Préstamo

Pueden:

  • Realizar todas las transacciones relacionadas con el servicio
  • Obtener reportes y productos

Crear Nuevo Operador

Haga clic sobre Nuevo usuario para crear un nuevo operador de ABCD. Se presentará una hoja de entrada con los campos de información requeridos. Complete la información solicitada y haga clic sobre:

  • Guardar el registro, o
  • Cancelar

según la acción requerida. [inicio]

Editar Operador

Haga clic sobre el icono de edición asociado al operador cuya información desea editar. Se presentará el formulario de ingreso con la información correspondiente. Edite el registro y al finalizar haga clic sobre guardar o cancelar dependiendo de la acción requerida. [inicio]

Eliminar Operador

Haga clic sobre el icono de eliminación asociado al operador que desea eliminar. [inicio]
