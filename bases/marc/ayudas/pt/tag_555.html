CAMPO 555 - NOTA DE ÍNDICE CUMULATIVO E REMISSIVO(R) Este campo contém informação identificando a disponibilidade de índice cumulativo e/ou remissivo cujo único ou maior enfoque é o material descrito. Para periódicos, este campo contém informação de volume e/ou datas cobertas pelo índice cumulativo de um item e informação de localização desses índices. Os índices podem ser editados como parte do periódico ou separadamente. Indicador 1 - Controle de constante para visualização branco não há informação - Índice, pode ser gerado 0 índice remissivo 8 não gerar constante para visualização Indicador 2 - Indefinido; contém branco (#) Códigos de subcampo \a nota de índice cumulativo e remissivo \b fonte disponível Exemplo: 555 |# #| \aInclui índice cumulativo no v.3
