CAMPO 256 – CARACTERÍSTICAS DO ARQUIVO DE COMPUTADOR (NR) Este campo é usado para registrar características de arquivos de computador. Pode conter informação sobre o tipo do arquivo (exemplo: Programas de computador), o número de registros, configuração, etc. (exemplo: 1250 registros, 5076 bytes). Indicadores – Indefinidos; cada um contém branco (#) Códigos de subcampo \a características de arquivo de computador Exemplo: 256 |# #| \aPrograma de computador (2 arquivos: 1200 registros, 4350 bytes)
