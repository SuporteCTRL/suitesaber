CAMPO 534 - NOTA DE VERSÃO ORIGINAL (R) Utilizar para descrever a produção original de uma obra. A reprodução é descrita na parte principal do registro bibliográfico. Pontuação: Para separar a frase introdutória do título (ambos no subcampo \a) utilize espaço-dois pontos-espaço. Para separar o título original (subcampo \a) dos dados de publicação use ponto finalespaço. As informações do subcampo \c devem seguir as regras de pontuação do campo 260. Indicadores – Indefinidos; cada um contém branco (#) Códigos de subcampo \a entrada principal do original \b edição do original \c publicação, distribuição, etc. do original \p frase introdutória \t título do original Observar seqüência: \p \t \b \c Exemplo: 534 |# #| \pFac-símile de: \tViagens na nossa terra. \cRio de Janeiro : A Noite, 1938
