Notas :
  • Esta opción busca palabras en nombres de instituciones, tipo de norma, y tema de la norma. Para buscar por número de norma use el formulario de búsqueda experta.
  • Use el símbolo de truncar $ para buscar palabras con la misma raíz.
    Ejemplo: econ$ recupera economía, economías, económico, etc. No es necesario usar operadores booleanos (AND, OR o AND NOT) entre las palabras. Seleccione la opción todas las palabras (AND) para relacionar las palabras (restringe la búsqueda), o la opción alguna palabra (OR) para sumar las palabras (amplia la búsqueda).
  • Para buscar por los números de las normas, o cualquier otra numeración use el formulario de Búsqueda avanzada.
  • Para buscar por otros campos de datos o para especificar el campo de busqueda, use el formulario de Búsqueda experta.
